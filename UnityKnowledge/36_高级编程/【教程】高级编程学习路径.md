# 高级编程学习路径

> 目标：掌握 C# 高级特性和高性能编程技术

## 前置知识

- C# 基础语法
- 面向对象编程
- 泛型基础

---

## 学习路线

```
Week 1-2: C# 高级特性
    ├── async/await 与状态机
    ├── 反射与元数据
    └── Expression 表达式树

Week 3-4: 并发编程
    ├── 线程与线程池
    ├── 锁与无锁编程
    └── async/await 最佳实践

Week 5-6: 高性能编程
    ├── Span/Memory/ArrayPool
    ├── 缓存友好代码
    └── SIMD 与向量化
```

---

## 核心知识点

### 1. async/await 原理

| 概念 | 说明 |
|------|------|
| 状态机 | 编译器生成的 IAsyncStateMachine |
| Task | 异步操作的表示 |
| CancellationToken | 取消异步操作 |
| ConfigureAwait | 同步上下文控制 |

### 2. 反射与元数据

| 概念 | 说明 |
|------|------|
| Type | 类型元数据 |
| PropertyInfo/FieldInfo | 成员信息 |
| MethodInfo | 方法信息 |
| Attribute | 自定义特性 |
| Expression | 表达式树 |

### 3. 内存与性能

| 概念 | 说明 |
|------|------|
| Span<T> | 内存切片，零分配 |
| Memory<T> | 可存储的内存引用 |
| ArrayPool<T> | 数组池化 |
| stackalloc | 栈上分配 |
| GC.Mode | 垃圾回收模式 |

---

## 实践项目

1. **对象映射器** - 使用 Expression 实现高性能对象映射
2. **异步任务管理器** - 管理大量并发异步任务
3. **高性能缓冲池** - 使用 ArrayPool 实现零GC缓冲
