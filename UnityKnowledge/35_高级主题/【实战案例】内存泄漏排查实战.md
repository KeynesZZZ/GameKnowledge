# 内存泄漏排查实战

> 专题课程 | 性能优化实战

## 1. 内存泄漏基础

### 1.1 什么是内存泄漏

```
┌─────────────────────────────────────────────────────────────┐
│                   内存泄漏定义                               │
│                                                             │
│  内存泄漏：程序运行过程中，已分配的内存无法被回收             │
│                                                             │
│  Unity 中的两类泄漏：                                        │
│                                                             │
│  1. 托管内存泄漏 (Managed Memory Leak)                      │
│     ├── C# 对象被引用，无法被 GC 回收                        │
│     ├── 常见原因：事件未取消、静态集合、闭包捕获             │
│     └── 症状：GC Heap 持续增长                               │
│                                                             │
│  2. 原生内存泄漏 (Native Memory Leak)                       │
│     ├── Unity 底层资源未被释放                               │
│     ├── 常见原因：纹理、网格、AssetBundle 未卸载             │
│     └── 症状：Profiler 显示 Native 内存持续增长              │
│                                                             │
│  3. 显存泄漏 (GPU Memory Leak)                              │
│     ├── RenderTexture 未释放                                │
│     ├── ComputeBuffer 未释放                                │
│     └── 症状：显存占用持续增长，最终崩溃                      │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 内存泄漏的症状

```csharp
using UnityEngine;

/// <summary>
/// 内存泄漏常见症状检测
/// </summary>
public class MemoryLeakSymptoms : MonoBehaviour
{
    /*
    ========== 症状清单 ==========

    1. 游戏运行时间越长越卡
       → GC 频繁触发，堆内存碎片化

    2. 切换场景后内存不下降
       → 资源未卸载，引用未释放

    3. Profiler 内存曲线持续上升
       → 存在持续分配未回收

    4. 设备发热严重
       → GC 和内存分配频繁

    5. 最终崩溃 (OOM)
       → 内存耗尽

    ========== 快速检测方法 ==========

    1. 运行游戏 10-30 分钟
    2. 观察 Profiler > Memory 面板
    3. 执行重复操作（进出场景、战斗等）
    4. 检查内存是否只增不减
    */
}
```

## 2. 排查工具详解

### 2.1 Unity Profiler

```csharp
using UnityEngine;
using UnityEngine.Profiling;

/// <summary>
/// Profiler 内存监控工具
/// </summary>
public class ProfilerMemoryMonitor : MonoBehaviour
{
    [Header("Settings")]
    [SerializeField] private bool autoLog = true;
    [SerializeField] private float logInterval = 5f;

    private float nextLogTime;

    private void Update()
    {
        if (autoLog && Time.time >= nextLogTime)
        {
            LogMemoryStatus();
            nextLogTime = Time.time + logInterval;
        }
    }

    /// <summary>
    /// 输出详细内存状态
    /// </summary>
    public static void LogMemoryStatus()
    {
        Debug.Log("========== Memory Status ==========");

        // === 托管内存 ===
        long gcMemory = System.GC.GetTotalMemory(false);
        Debug.Log($"[Managed] GC Heap: {FormatBytes(gcMemory)}");

        // === Unity 内存 ===
        long totalAllocated = Profiler.GetTotalAllocatedMemoryLong();
        long totalReserved = Profiler.GetTotalReservedMemoryLong();
        long totalUnusedReserved = Profiler.GetTotalUnusedReservedMemoryLong();

        Debug.Log($"[Unity] Allocated: {FormatBytes(totalAllocated)}");
        Debug.Log($"[Unity] Reserved: {FormatBytes(totalReserved)}");
        Debug.Log($"[Unity] Unused Reserved: {FormatBytes(totalUnusedReserved)}");

        // === 各类资源统计 ===
        Debug.Log($"[Textures] Count: {Resources.FindObjectsOfTypeAll<Texture>().Length}");
        Debug.Log($"[Meshes] Count: {Resources.FindObjectsOfTypeAll<Mesh>().Length}");
        Debug.Log($"[Materials] Count: {Resources.FindObjectsOfTypeAll<Material>().Length}");
        Debug.Log($"[AudioClips] Count: {Resources.FindObjectsOfTypeAll<AudioClip>().Length}");

        // === GameObject 统计 ===
        var allGOs = FindObjectsOfType<GameObject>();
        Debug.Log($"[GameObjects] Active: {allGOs.Length}");

        Debug.Log("====================================");
    }

    /// <summary>
    /// 检测内存增长趋势
    /// </summary>
    public class MemoryTrendTracker
    {
        private long lastMemory;
        private int increaseCount;
        private int decreaseCount;

        public void Check()
        {
            long currentMemory = System.GC.GetTotalMemory(false);
            long delta = currentMemory - lastMemory;

            if (delta > 1024 * 1024) // 增长超过 1MB
            {
                increaseCount++;
                Debug.LogWarning($"[Memory Trend] Increased by {FormatBytes(delta)} " +
                               $"(consecutive increases: {increaseCount})");
            }
            else if (delta < -1024 * 1024) // 下降超过 1MB
            {
                decreaseCount++;
                increaseCount = 0; // 重置连续增长计数
            }

            lastMemory = currentMemory;

            // 连续 5 次增长，发出警告
            if (increaseCount >= 5)
            {
                Debug.LogError("[Memory Leak Warning] Memory has been increasing for 5 consecutive checks!");
            }
        }
    }

    private static string FormatBytes(long bytes)
    {
        string[] sizes = { "B", "KB", "MB", "GB" };
        int order = 0;
        double size = bytes;

        while (size >= 1024 && order < sizes.Length - 1)
        {
            order++;
            size /= 1024;
        }

        return $"{size:0.##} {sizes[order]}";
    }
}
```

### 2.3 自定义泄漏检测器

```csharp
using UnityEngine;
using System.Collections.Generic;
using System;

/// <summary>
/// 高级内存泄漏检测器
/// </summary>
public class AdvancedLeakDetector : MonoBehaviour
{
    private static AdvancedLeakDetector instance;
    public static AdvancedLeakDetector Instance
    {
        get
        {
            if (instance == null)
            {
                var go = new GameObject("[LeakDetector]");
                instance = go.AddComponent<AdvancedLeakDetector>();
                DontDestroyOnLoad(go);
            }
            return instance;
        }
    }

    [Header("Settings")]
    [SerializeField] private float checkInterval = 10f;
    [SerializeField] private long warningThresholdBytes = 10 * 1024 * 1024; // 10MB
    [SerializeField] private int consecutiveIncreaseWarning = 3;

    // 内存历史
    private List<MemorySnapshot> memoryHistory = new List<MemorySnapshot>();
    private int consecutiveIncreases = 0;

    // 对象追踪
    private Dictionary<Type, int> objectCounts = new Dictionary<Type, int>();
    private Dictionary<Type, int> previousCounts = new Dictionary<Type, int>();

    // 关注的类型（可配置）
    private Type[] watchTypes = new Type[]
    {
        typeof(Texture2D),
        typeof(Mesh),
        typeof(Material),
        typeof(AudioClip),
        typeof(GameObject),
        typeof(MonoBehaviour)
    };

    private struct MemorySnapshot
    {
        public long timestamp;
        public long gcMemory;
        public long nativeMemory;
        public Dictionary<Type, int> objectCounts;
    }

    private void Start()
    {
        StartCoroutine(CheckRoutine());
    }

    private System.Collections.IEnumerator CheckRoutine()
    {
        while (true)
        {
            yield return new WaitForSeconds(checkInterval);
            PerformCheck();
        }
    }

    private void PerformCheck()
    {
        // 记录当前内存
        var snapshot = new MemorySnapshot
        {
            timestamp = DateTime.Now.Ticks,
            gcMemory = GC.GetTotalMemory(false),
            nativeMemory = UnityEngine.Profiling.Profiler.GetTotalAllocatedMemoryLong(),
            objectCounts = CaptureObjectCounts()
        };

        memoryHistory.Add(snapshot);

        // 只保留最近 10 次记录
        if (memoryHistory.Count > 10)
            memoryHistory.RemoveAt(0);

        // 分析趋势
        AnalyzeTrend(snapshot);

        // 检测对象增长
        DetectObjectGrowth(snapshot.objectCounts);
    }

    private Dictionary<Type, int> CaptureObjectCounts()
    {
        var counts = new Dictionary<Type, int>();

        foreach (var type in watchTypes)
        {
            var objects = Resources.FindObjectsOfTypeAll(type);
            counts[type] = objects.Length;
        }

        return counts;
    }

    private void AnalyzeTrend(MemorySnapshot current)
    {
        if (memoryHistory.Count < 2) return;

        var previous = memoryHistory[memoryHistory.Count - 2];
        long gcDelta = current.gcMemory - previous.gcMemory;
        long nativeDelta = current.nativeMemory - previous.nativeMemory;

        Debug.Log($"[Leak Check] GC: {FormatBytes(gcDelta:+#;-#;0)}, " +
                 $"Native: {FormatBytes(nativeDelta:+#;-#;0)}");

        // 检查连续增长
        if (gcDelta > warningThresholdBytes || nativeDelta > warningThresholdBytes)
        {
            consecutiveIncreases++;

            if (consecutiveIncreases >= consecutiveIncreaseWarning)
            {
                Debug.LogError($"[LEAK WARNING] Memory increased for {consecutiveIncreases} consecutive checks!");
                GenerateLeakReport();
            }
        }
        else
        {
            consecutiveIncreases = 0;
        }
    }

    private void DetectObjectGrowth(Dictionary<Type, int> currentCounts)
    {
        foreach (var kvp in currentCounts)
        {
            int previousCount = previousCounts.GetValueOrDefault(kvp.Key, 0);
            int delta = kvp.Value - previousCount;

            if (delta > 5) // 显著增长
            {
                Debug.LogWarning($"[Object Growth] {kvp.Key.Name}: +{delta} " +
                               $"({previousCount} → {kvp.Value})");
            }
        }

        previousCounts = new Dictionary<Type, int>(currentCounts);
    }

    private void GenerateLeakReport()
    {
        Debug.Log("========== LEAK REPORT ==========");

        // 1. 检查事件订阅
        Debug.Log("[1] Checking event subscriptions...");

        // 2. 检查静态集合
        Debug.Log("[2] Checking static collections...");

        // 3. 检查协程
        Debug.Log("[3] Checking active coroutines...");

        // 4. 资源列表
        Debug.Log("[4] Resource summary:");
        foreach (var type in watchTypes)
        {
            var objects = Resources.FindObjectsOfTypeAll(type);
            Debug.Log($"  {type.Name}: {objects.Length}");
        }

        Debug.Log("=================================");
    }

    /// <summary>
    /// 手动触发检查
    /// </summary>
    public static void ForceCheck()
    {
        Instance?.PerformCheck();
    }

    /// <summary>
    /// 重置计数器
    /// </summary>
    public static void ResetCounter()
    {
        if (Instance != null)
        {
            Instance.consecutiveIncreases = 0;
            Instance.memoryHistory.Clear();
            Instance.previousCounts.Clear();
        }
    }

    private static string FormatBytes(long bytes)
    {
        if (bytes < 0) return $"-{FormatBytes(-bytes)}";

        string[] sizes = { "B", "KB", "MB", "GB" };
        int order = 0;
        double size = bytes;

        while (size >= 1024 && order < sizes.Length - 1)
        {
            order++;
            size /= 1024;
        }

        return $"{size:0.##} {sizes[order]}";
    }
}
```

## 3. 常见泄漏模式详解

### 3.1 事件订阅泄漏

```csharp
using UnityEngine;
using System;

/// <summary>
/// 事件订阅泄漏示例与解决方案
/// </summary>
public class EventLeakExample : MonoBehaviour
{
    // ========== 泄漏示例 ==========

    // 错误：在 OnEnable 订阅，但没有在 OnDisable 取消
    private void OnEnable_Bad()
    {
        // 每次启用都添加新的订阅，但从不移除
        EventManager.OnPlayerDeath += HandlePlayerDeath;
        EventManager.OnScoreChanged += HandleScoreChanged;
    }

    // 错误：使用 lambda 表达式
    private void Start_Bad()
    {
        // lambda 无法取消订阅！
        EventManager.OnScoreChanged += (score) => Debug.Log(score);
    }

    // ========== 正确做法 ==========

    private void OnEnable()
    {
        EventManager.OnPlayerDeath += HandlePlayerDeath;
        EventManager.OnScoreChanged += HandleScoreChanged;
    }

    private void OnDisable()
    {
        // 必须在 OnDisable 或 OnDestroy 中取消订阅
        EventManager.OnPlayerDeath -= HandlePlayerDeath;
        EventManager.OnScoreChanged -= HandleScoreChanged;
    }

    private void HandlePlayerDeath() { }
    private void HandleScoreChanged(int score) { }

    // ========== 使用 WeakReference 防止泄漏 ==========

    public class WeakEventManager
    {
        private static List<WeakReference<Action>> weakActions = new List<WeakReference<Action>>();

        public static void SubscribeWeak(Action action)
        {
            weakActions.Add(new WeakReference<Action>(action));
        }

        public static void Invoke()
        {
            // 清理已释放的引用
            weakActions.RemoveAll(wr => !wr.TryGetTarget(out _));

            foreach (var wr in weakActions)
            {
                if (wr.TryGetTarget(out var action))
                    action?.Invoke();
            }
        }
    }

    // ========== 事件订阅追踪工具 ==========

    public static class EventSubscriptionTracker
    {
        private static Dictionary<string, int> subscriptionCounts = new Dictionary<string, int>();

        public static void TrackSubscribe(string eventName)
        {
            if (!subscriptionCounts.ContainsKey(eventName))
                subscriptionCounts[eventName] = 0;
            subscriptionCounts[eventName]++;

            Debug.Log($"[Event] {eventName} subscriptions: {subscriptionCounts[eventName]}");
        }

        public static void TrackUnsubscribe(string eventName)
        {
            if (subscriptionCounts.ContainsKey(eventName))
                subscriptionCounts[eventName]--;

            Debug.Log($"[Event] {eventName} subscriptions: {subscriptionCounts[eventName]}");
        }

        public static void Report()
        {
            Debug.Log("===== Event Subscription Report =====");
            foreach (var kvp in subscriptionCounts)
            {
                if (kvp.Value != 0)
                    Debug.LogWarning($"[Event Leak] {kvp.Key}: {kvp.Value} unbalanced subscriptions!");
            }
            Debug.Log("=====================================");
        }
    }
}
```

### 3.2 静态集合泄漏

```csharp
using UnityEngine;
using System.Collections.Generic;

/// <summary>
/// 静态集合泄漏示例与解决方案
/// </summary>
public class StaticCollectionLeak : MonoBehaviour
{
    // ========== 泄漏示例 ==========

    // 错误：静态列表只增不减
    public static class EnemyManager_Bad
    {
        public static List<Enemy> allEnemies = new List<Enemy>();

        public static void Register(Enemy enemy)
        {
            allEnemies.Add(enemy);
        }
        // 忘记提供 Unregister 方法！
    }

    // ========== 正确做法 ==========

    public static class EnemyManager
    {
        private static List<Enemy> allEnemies = new List<Enemy>();

        public static void Register(Enemy enemy)
        {
            if (!allEnemies.Contains(enemy))
                allEnemies.Add(enemy);
        }

        public static void Unregister(Enemy enemy)
        {
            allEnemies.Remove(enemy);
        }

        public static int Count => allEnemies.Count;

        // 调试方法：检查是否有已销毁的对象
        public static void CheckForDestroyed()
        {
            int removed = allEnemies.RemoveAll(e => e == null);
            if (removed > 0)
            {
                Debug.LogWarning($"[Static Collection] Removed {removed} destroyed enemies");
            }
        }
    }

    // ========== 敌人类正确实现 ==========

    public class Enemy : MonoBehaviour
    {
        private void OnEnable()
        {
            EnemyManager.Register(this);
        }

        private void OnDisable()
        {
            EnemyManager.Unregister(this);
        }
    }

    // ========== 使用 WeakReference 的静态集合 ==========

    public static class WeakEnemyTracker
    {
        private static List<WeakReference<Enemy>> weakEnemies = new List<WeakReference<Enemy>>();

        public static void Track(Enemy enemy)
        {
            weakEnemies.Add(new WeakReference<Enemy>(enemy));
        }

        public static void Cleanup()
        {
            // 移除已被 GC 的对象
            weakEnemies.RemoveAll(wr =>
            {
                return !wr.TryGetTarget(out _);
            });
        }

        public static int GetAliveCount()
        {
            Cleanup();
            return weakEnemies.Count;
        }
    }

    // ========== 静态集合检测工具 ==========

    public static class StaticCollectionChecker
    {
        public static void CheckAllStaticCollections()
        {
            Debug.Log("===== Static Collection Check =====");

            // 使用反射查找所有静态集合
            var assemblies = System.AppDomain.CurrentDomain.GetAssemblies();

            foreach (var assembly in assemblies)
            {
                try
                {
                    foreach (var type in assembly.GetTypes())
                    {
                        foreach (var field in type.GetFields(
                            System.Reflection.BindingFlags.Static |
                            System.Reflection.BindingFlags.Public |
                            System.Reflection.BindingFlags.NonPublic))
                        {
                            if (typeof(System.Collections.ICollection).IsAssignableFrom(field.FieldType))
                            {
                                var value = field.GetValue(null) as System.Collections.ICollection;
                                if (value != null && value.Count > 0)
                                {
                                    Debug.Log($"[Static] {type.Name}.{field.Name}: {value.Count} items");
                                }
                            }
                        }
                    }
                }
                catch { }
            }

            Debug.Log("====================================");
        }
    }
}
```

### 3.3 协程泄漏

```csharp
using UnityEngine;
using System.Collections;

/// <summary>
/// 协程泄漏示例与解决方案
/// </summary>
public class CoroutineLeak : MonoBehaviour
{
    // ========== 泄漏示例 ==========

    // 错误：启动协程但从不停止
    private void Start_Bad()
    {
        StartCoroutine(InfiniteLoop());
    }

    private IEnumerator InfiniteLoop()
    {
        while (true)
        {
            // 每帧分配内存
            string message = "Frame: " + Time.frameCount;
            Debug.Log(message);
            yield return null;
        }
    }

    // 错误：在协程中捕获 this
    private void Start_Bad2()
    {
        StartCoroutine(CoroutineWithCapture());
    }

    private IEnumerator CoroutineWithCapture()
    {
        yield return new WaitForSeconds(1f);
        // 即使对象被销毁，协程仍然持有引用
        transform.position = Vector3.zero; // 可能报错或泄漏
    }

    // ========== 正确做法 ==========

    private Coroutine currentCoroutine;

    private void Start()
    {
        currentCoroutine = StartCoroutine(SafeCoroutine());
    }

    private void OnDestroy()
    {
        // 销毁时停止协程
        if (currentCoroutine != null)
        {
            StopCoroutine(currentCoroutine);
            currentCoroutine = null;
        }
    }

    private IEnumerator SafeCoroutine()
    {
        while (this != null && gameObject.activeInHierarchy)
        {
            // 检查对象是否有效
            yield return null;
        }
    }

    // ========== 协程管理器 ==========

    public class CoroutineManager : MonoBehaviour
    {
        private static CoroutineManager instance;
        public static CoroutineManager Instance
        {
            get
            {
                if (instance == null)
                {
                    var go = new GameObject("[CoroutineManager]");
                    instance = go.AddComponent<CoroutineManager>();
                    DontDestroyOnLoad(go);
                }
                return instance;
            }
        }

        // 追踪所有协程
        private Dictionary<string, Coroutine> trackedCoroutines = new Dictionary<string, Coroutine>();

        /// <summary>
        /// 启动带追踪的协程
        /// </summary>
        public void StartTrackedCoroutine(string key, IEnumerator routine)
        {
            // 如果已存在同名协程，先停止
            StopTrackedCoroutine(key);

            trackedCoroutines[key] = StartCoroutine(routine);
            Debug.Log($"[Coroutine] Started: {key}");
        }

        /// <summary>
        /// 停止追踪的协程
        /// </summary>
        public void StopTrackedCoroutine(string key)
        {
            if (trackedCoroutines.TryGetValue(key, out var coroutine))
            {
                if (coroutine != null)
                    StopCoroutine(coroutine);

                trackedCoroutines.Remove(key);
                Debug.Log($"[Coroutine] Stopped: {key}");
            }
        }

        /// <summary>
        /// 停止所有追踪的协程
        /// </summary>
        public void StopAllTracked()
        {
            foreach (var kvp in trackedCoroutines)
            {
                if (kvp.Value != null)
                    StopCoroutine(kvp.Value);
            }

            Debug.Log($"[Coroutine] Stopped all: {trackedCoroutines.Count}");
            trackedCoroutines.Clear();
        }

        /// <summary>
        /// 输出协程报告
        /// </summary>
        public void Report()
        {
            Debug.Log($"[Coroutine] Active count: {trackedCoroutines.Count}");
            foreach (var key in trackedCoroutines.Keys)
            {
                Debug.Log($"  - {key}");
            }
        }
    }

    // ========== 使用示例 ==========

    public class CoroutineExample : MonoBehaviour
    {
        private void Start()
        {
            // 使用管理器启动协程
            CoroutineManager.Instance.StartTrackedCoroutine("AutoSave", AutoSaveRoutine());
            CoroutineManager.Instance.StartTrackedCoroutine("Spawn", SpawnRoutine());
        }

        private void OnDestroy()
        {
            // 停止特定协程
            CoroutineManager.Instance.StopTrackedCoroutine("AutoSave");
            CoroutineManager.Instance.StopTrackedCoroutine("Spawn");
        }

        private IEnumerator AutoSaveRoutine()
        {
            while (true)
            {
                yield return new WaitForSeconds(60f);
                // 保存游戏
            }
        }

        private IEnumerator SpawnRoutine()
        {
            while (true)
            {
                yield return new WaitForSeconds(1f);
                // 生成敌人
            }
        }
    }
}
```

```csharp
    private void LoadAssetBundleGood(string path)
    {
        if (loadedBundles.ContainsKey(path)) return;

        var bundle = AssetBundle.LoadFromFile(path);
        loadedBundles[path] = bundle;
    }

    private void UnloadAssetBundle(string path, bool unloadAllLoadedObjects)
    {
        if (loadedBundles.TryGetValue(path, out var bundle))
        {
            bundle.Unload(unloadAllLoadedObjects);
            loadedBundles.Remove(path);
        }

    // ========== 资源追踪工具 ==========

    public static class ResourceTracker
    {
        private static Dictionary<string, List<Object>> trackedResources = new Dictionary<string, List<Object>>();

        /// <summary>
        /// 追踪资源
        /// </summary>
        public static void Track(string category, Object resource)
        {
            if (!trackedResources.ContainsKey(category))
                trackedResources[category] = new List<Object>();

            trackedResources[category].Add(resource);
        }

        /// <summary>
        /// 卸载分类下的所有资源
        /// </summary>
        public static void UnloadCategory(string category)
        {
            if (trackedResources.TryGetValue(category, out var resources))
            {
                foreach (var res in resources)
                {
                    if (res != null)
                    {
                        if (res is Texture || res is Mesh || res is AudioClip)
                            Resources.UnloadAsset(res);
                        else if (res is Material || res is GameObject)
                            Object.Destroy(res);
                    }
                }
                resources.Clear();
                Debug.Log($"[ResourceTracker] Unloaded category: {category}");
            }
        }

        /// <summary>
        /// 输出报告
        /// </summary>
        public static void Report()
        {
            Debug.Log("===== Resource Tracker Report =====");
            foreach (var kvp in trackedResources)
            {
                // 过滤掉已销毁的对象
                kvp.Value.RemoveAll(obj => obj == null);
                Debug.Log($"[{kvp.Key}] Count: {kvp.Value.Count}");
            }
            Debug.Log("===================================");
        }

        /// <summary>
        /// 清理所有追踪
        /// </summary>
        public static void ClearAll()
        {
            foreach (var category in trackedResources.Keys)
            {
                UnloadCategory(category);
            }
            trackedResources.Clear();
        }
    }

    // ========== RenderTexture 泄漏检测 ==========

    public static class RenderTextureLeakDetector
    {
        private static int lastRTCount;

        public static void Check()
        {
            var allRTs = Resources.FindObjectsOfTypeAll<RenderTexture>();
            int currentCount = allRTs.Length;

            if (currentCount > lastRTCount)
            {
                Debug.LogWarning($"[RenderTexture Leak] Count increased: {lastRTCount} → {currentCount}");
                foreach (var rt in allRTs)
                {
                    Debug.Log($"  - {rt.name}: {rt.width}x{rt.height} {rt.format}");
                }
            }

            lastRTCount = currentCount;
        }
    }
```

```csharp
using UnityEngine;
using System;

/// <summary>
/// 闭包泄漏示例与解决方案
/// </summary>
public class ClosureLeak : MonoBehaviour
{
    // ========== 泄漏示例 ==========

    // 错误：闭包捕获 this，导致无法 GC
    private void AddListener_Bad()
    {
        button.onClick.AddListener(() =>
        {
            // 闭包捕获了 this，即使对象销毁，委托仍持有引用
            UpdateUI(this.data);
        });
    }

    // 错误：闭包捕获大对象
    private void StartCoroutine_Bad()
    {
        LargeData largeData = GetLargeData();
        StartCoroutine(ProcessData(largeData));
    }

    private IEnumerator ProcessData(LargeData data)
    {
        yield return new WaitForSeconds(1f);
        // 协程持有 largeData 引用，即使不再需要
        Process(data);
    }

    // ========== 正确做法 ==========

    private UnityEngine.UI.Button button;
    private string data;

    // 方案 1：使用方法引用
    private void AddListenerGood1()
    {
        button.onClick.AddListener(OnButtonClick);
    }

    private void OnButtonClick()
    {
        UpdateUI(data);
    }

    // 方案 2：使用弱引用
    private void AddListenerGood2()
    {
        var weakThis = new WeakReference<ClosureLeak>(this);
        button.onClick.AddListener(() =>
        {
            if (weakThis.TryGetTarget(out var target))
            {
                target.UpdateUI(target.data);
            }
        });
    }

    // 方案 3：及时取消订阅
    private Action cachedAction;

    private void OnEnable()
    {
        cachedAction = () => UpdateUI(data);
        button.onClick.AddListener(cachedAction);
    }

    private void OnDisable()
    {
        button.onClick.RemoveListener(cachedAction);
    }

    private void UpdateUI(string data) { }
    private LargeData GetLargeData() => null;
    private void Process(LargeData data) { }

    private class LargeData { }
}

// ========== 闭包检测工具 ==========

public static class ClosureDetector
{
    /*
    检测闭包捕获的方法：

    1. 使用 Memory Profiler 查看闭包对象
    2. 检查是否有 CompilerGenerated 类
    3. 查看委托的 Target 属性

    常见闭包泄漏模式：
    - lambda 表达式在事件订阅中
    - 协程中捕获变量
    - LINQ 表达式捕获外部变量
    - 匿名方法捕获 this
    */

    public static void AnalyzeDelegate(Delegate del)
    {
        if (del == null) return;

        Debug.Log($"[Delegate] Method: {del.Method.Name}");
        Debug.Log($"[Delegate] Target: {del.Target?.GetType().Name ?? "static"}");

        if (del.Target != null)
        {
            // 检查是否是编译器生成的闭包类
            var type = del.Target.GetType();
            if (type.Name.Contains("<") || type.Name.Contains("__"))
            {
                Debug.LogWarning($"[Closure] Detected compiler-generated closure: {type.Name}");
            }
        }
    }
}
```

### 4.2 一键检测脚本

```csharp
using UnityEngine;
using System.Collections.Generic;
using System;

/// <summary>
/// 内存泄漏一键检测工具
/// </summary>
public class MemoryLeakScanner : MonoBehaviour
{
    [MenuItem("Tools/Memory/Scan For Leaks")]
    public static void ScanForLeaks()
    {
        Debug.Log("╔═══════════════════════════════════════════════════════════╗");
        Debug.Log("║              Memory Leak Scanner v1.0                     ║");
        Debug.Log("╚═══════════════════════════════════════════════════════════╝");

        // 1. 检查内存状态
        CheckMemoryStatus();

        // 2. 检查资源
        CheckResources();

        // 3. 检查常见泄漏模式
        CheckCommonLeaks();

        // 4. 生成报告
        GenerateReport();
    }

    private static void CheckMemoryStatus()
    {
        Debug.Log("\n[1] Memory Status:");

        long gcMemory = GC.GetTotalMemory(false);
        long nativeMemory = UnityEngine.Profiling.Profiler.GetTotalAllocatedMemoryLong();

        Debug.Log($"    GC Heap: {FormatBytes(gcMemory)}");
        Debug.Log($"    Native Memory: {FormatBytes(nativeMemory)}");

        if (gcMemory > 500 * 1024 * 1024) // 500MB
        {
            Debug.LogWarning("    ⚠️ GC Heap is high! Possible managed memory leak.");
        }
    }

    private static void CheckResources()
    {
        Debug.Log("\n[2] Resource Count:");

        var textures = Resources.FindObjectsOfTypeAll<Texture>();
        var meshes = Resources.FindObjectsOfTypeAll<Mesh>();
        var materials = Resources.FindObjectsOfTypeAll<Material>();
        var audioClips = Resources.FindObjectsOfTypeAll<AudioClip>();
        var renderTextures = Resources.FindObjectsOfTypeAll<RenderTexture>();

        Debug.Log($"    Textures: {textures.Length}");
        Debug.Log($"    Meshes: {meshes.Length}");
        Debug.Log($"    Materials: {materials.Length}");
        Debug.Log($"    AudioClips: {audioClips.Length}");
        Debug.Log($"    RenderTextures: {renderTextures.Length}");

        // 检查大纹理
        int largeTextures = 0;
        foreach (var tex in textures)
        {
            if (tex.width > 2048 || tex.height > 2048)
            {
                Debug.LogWarning($"    ⚠️ Large texture: {tex.name} ({tex.width}x{tex.height})");
                largeTextures++;
            }
        }

        if (largeTextures > 0)
            Debug.LogWarning($"    Found {largeTextures} textures larger than 2048!");
    }

    private static void CheckCommonLeaks()
    {
        Debug.Log("\n[3] Common Leak Patterns:");

        // 检查 UI Raycast Targets
        var graphics = Resources.FindObjectsOfTypeAll<Graphic>();
        int raycastEnabled = 0;
        foreach (var g in graphics)
        {
            if (g.raycastTarget) raycastEnabled++;
        }
        float ratio = graphics.Length > 0 ? (float)raycastEnabled / graphics.Length : 0;
        Debug.Log($"    UI Raycast Targets: {raycastEnabled}/{graphics.Length} ({ratio:P0})");
        if (ratio > 0.5f)
            Debug.LogWarning("    ⚠️ More than 50% of Graphics have Raycast Target enabled!");

        // 检查协程
        var monoBehaviours = FindObjectsOfType<MonoBehaviour>();
        Debug.Log($"    Active MonoBehaviours: {monoBehaviours.Length}");

        // 检查 DontDestroyOnLoad 对象
        var ddonLoadObjects = FindObjectsOfType<GameObject>();
        int dontDestroyCount = 0;
        foreach (var go in ddonLoadObjects)
        {
            if (go.scene.name == "DontDestroyOnLoad")
                dontDestroyCount++;
        }
        Debug.Log($"    DontDestroyOnLoad Objects: {dontDestroyCount}");
    }

    private static void GenerateReport()
    {
        Debug.Log("\n[4] Recommendations:");

        Debug.Log("    □ Check event subscriptions in OnDisable/OnDestroy");
        Debug.Log("    □ Clear static collections when not needed");
        Debug.Log("    □ Stop coroutines in OnDestroy");
        Debug.Log("    □ Unload resources with Resources.UnloadAsset");
        Debug.Log("    □ Use Memory Profiler for detailed analysis");

        Debug.Log("\n╔═══════════════════════════════════════════════════════════╗");
        Debug.Log("║                    Scan Complete                          ║");
        Debug.Log("╚═══════════════════════════════════════════════════════════╝");
    }

    private static string FormatBytes(long bytes)
    {
        string[] sizes = { "B", "KB", "MB", "GB" };
        int order = 0;
        double size = bytes;

        while (size >= 1024 && order < sizes.Length - 1)
        {
            order++;
            size /= 1024;
        }

        return $"{size:0.##} {sizes[order]}";
    }
}
```

## 本课小结

### 核心知识点

| 知识点 | 核心要点 |
|--------|----------|
| 泄漏类型 | 托管内存、原生内存、显存 |
| 排查工具 | Profiler、Memory Profiler、自定义检测器 |
| 常见泄漏 | 事件订阅、静态集合、协程、资源、闭包 |
| 排查流程 | 确认→定位→分析→修复→验证 |

### 泄漏排查优先级

```
1. 事件订阅 (最常见)
   ├── 检查所有 += 是否有对应的 -=
   └── 确保 OnDisable 中取消订阅

2. 静态集合
   ├── 检查只增不减的 List/Dictionary
   └── 确保销毁时移除

3. 资源管理
   ├── 使用 Memory Profiler 找到未卸载资源
   └── 添加卸载逻辑

4. 协程
   ├── 检查无限循环协程
   └── 确保销毁时停止
```
