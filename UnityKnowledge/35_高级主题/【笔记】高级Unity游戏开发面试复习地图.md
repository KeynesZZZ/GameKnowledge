# 【笔记】高级Unity游戏开发面试复习地图

> 本页是一张**以「高级岗能力域」为骨架的复习导航图**，不是又一份题库。它的价值有三：① 把考点一一关联到 vault 已有笔记，点开即复习；② 标注高级岗的**重要度与考察形式**，告诉你先复习什么；③ 提供**自评 checklist** 查漏补缺。
>
> 与 [[【笔记】Unity客户端面试题]] 的分工：那份是**中级「题目+答案」速查**（算法 / Lua / 3D 数学为主）；本页是**高级岗「能力域+笔记导航+优先级」地图**，侧重中级之上的增量——**性能优化体系、渲染管线/Shader 深挖、热更新与资源管理、架构演进、项目经验组织**。两者配合使用。`#面试` `#综述`

## 文档定位

- **目标岗位**：北京 / 一线城市，高级 Unity 游戏客户端开发（3–5 年中级、5–10 年高级/资深、主程方向）。
- **JD 共性（2026-06 采集）**：精通 C# + Unity 引擎架构、有完整上线项目、**图形渲染与 Shader**、**性能优化**、**热更新（HybridCLR / Addressables）**、UGUI、设计模式是高频硬性要求；加分项 DOTS、编辑器扩展、AR/VR、C++。
- **使用方式**：按「复习优先级总览」排程 → 每个域过一遍考点 → 点关联笔记深读 → 用自评表打勾 → 项目经验用 STAR 整理话术。

## 十、项目经验 & 工程实践（必问，提前整理）

- 一个**性能优化**故事（指标前后对比、定位过程、最终方案）
- 一个**架构/技术选型**故事（为什么这么选、代价、演进）
- 一个**踩坑/攻坚**故事（根因分析、规避方法）

## 复习路线建议

- **2 周计划**：第 1 周——性能优化（域一）+ 渲染/Shader（域二）+ UGUI（域四），这三块占高级面试 60%+ 篇幅；第 2 周——热更新/资源（域三）+ 架构（域五）+ C#/动画（域六七）+ 项目话术整理（域十），算法/3D 数学（域八）每天穿插刷。
- **1 周冲刺**：只看带 ⭐⭐⭐ 的四个域 + 自评 Checklist + 项目 STAR，其余按下不表。
