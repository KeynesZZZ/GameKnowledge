# 【教程】AudioMixer使用指南

> AudioMixer 是音频系统真正进入“可控状态”的起点。

## 文档定位

本文档用于指导 AudioMixer 的基础搭建与项目落地方式，适合在项目开始统一音量分组、淡入淡出和效果链路时使用。

## 前置知识

- 了解 `AudioSource` 与 `AudioClip`
- 了解 BGM、SFX、UI 音效的分类方式
- 已具备一个基础的音频播放入口

## 步骤

1. 创建主 Mixer
2. 按类型建立 Group：Master / BGM / SFX / UI / Voice
3. 暴露关键参数：音量、低通、高通
4. 在设置系统中绑定滑杆到暴露参数
5. 为场景切换和暂停设计淡入淡出规则

## 完整示例

建议最小落地方案：

- 一个全局 `AudioMixer`
- 三个主要分组：`BGM`、`SFX`、`UI`
- 一个统一设置面板控制三类音量

## 常见错误

- 直接改 `AudioSource.volume`，导致全局设置难统一
- 没有分组，后期无法快速做音量隔离
- 参数命名不统一，脚本侧容易绑错

- 为战斗或 UI 场景设计不同的混音快照
- 结合加载系统统一管理音频资源生命周期
