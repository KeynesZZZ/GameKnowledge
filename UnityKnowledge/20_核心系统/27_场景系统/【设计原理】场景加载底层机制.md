## 快速参考

```csharp
// 同步加载场景
SceneManager.LoadScene("Level1", LoadSceneMode.Single);

// 异步加载场景
StartCoroutine(LoadSceneAsync("Level1", LoadSceneMode.Single));

IEnumerator LoadSceneAsync(string sceneName, LoadSceneMode mode)
{
    AsyncOperation asyncLoad = SceneManager.LoadSceneAsync(sceneName, mode);
    
    // 等待加载完成
    while (!asyncLoad.isDone)
    {
        // 显示加载进度 (0.0 - 0.9)
        float progress = Mathf.Clamp01(asyncLoad.progress / 0.9f);
        Debug.Log($"Loading: {progress * 100:F1}%");
        yield return null;
    }
    
    Debug.Log("Scene loaded!");
}

// 异步加载场景（带进度条）
StartCoroutine(LoadSceneWithProgress("Level1"));

IEnumerator LoadSceneWithProgress(string sceneName)
{
    var asyncLoad = SceneManager.LoadSceneAsync(sceneName, LoadSceneMode.Single);
    
    // 允许激活场景
    asyncLoad.allowSceneActivation = false;
    
    while (!asyncLoad.isDone)
    {
        // 进度 0.0 - 1.0
        float progress = asyncLoad.progress;
        Debug.Log($"Progress: {progress * 100:F1}%");
        yield return null;
    }
    
    asyncLoad.allowSceneActivation = true;
}
```

## 场景管理架构

### SceneManager核心组件

```
SceneManager (场景管理器)
│
├─> Scene (场景)
│   ├─> name (场景名称)
│   ├─> path (场景路径)
│   ├─> buildIndex (Build Index)
│   ├─> isLoaded (是否加载)
│   ├─> isDirty (是否修改)
│   └─> rootGameObjects (根游戏对象列表)
│
├─> Scene (场景列表)
│   ├─> Count (场景数量)
│   ├─> operator [] (索引器)
│   └─> GetEnumerator (枚举器)
│
├─> LoadSceneAsync (异步加载)
│   ├─> allowSceneActivation (是否激活)
│   ├─> isDone (是否完成)
│   ├─> progress (进度 0.0-1.0)
│   └─> priority (优先级)
│
└─> UnloadSceneAsync (异步卸载)
    ├─> isDone (是否完成)
    └─> progress (进度 0.0-1.0)
```

### 场景生命周期

```
场景生命周期流程:

1. 加载阶段 (Loading)
   ├─> 读取场景文件
   ├─> 解析场景数据
   ├─> 反序列化游戏对象
   └─> 加载资源和引用

2. 激活阶段 (Activation)
   ├─> 创建游戏对象
   ├─> 调用Awake()
   ├─> 调用OnEnable()
   └─> 调用Start()

3. 运行阶段 (Running)
   ├─> Update()
   ├─> LateUpdate()
   └─> FixedUpdate()

4. 卸载阶段 (Unloading)
   ├─> 调用OnDisable()
   ├─> 调用OnDestroy()
   ├─> 销毁游戏对象
   └─> 释放资源
```

## 场景加载模式

### 1. Single 模式 (单场景)

```
Single模式流程:

加载前: [SceneA]
         ↓ LoadScene("SceneB", Single)
加载中: [SceneA → SceneB] (过渡期)
加载后: [SceneB]

特点:
├─> 卸载所有已加载场景
├─> 加载新场景
├─> 简单直接
└─> 适用于游戏关卡切换
```

**应用场景：**
- 关卡切换
- 游戏开始/结束
- 主菜单进入游戏
- 单关卡游戏

### 2. Additive 模式 (叠加加载)

```
Additive模式流程:

加载前: [SceneA]
         ↓ LoadScene("SceneB", Additive)
加载中: [SceneA, SceneB] (叠加)
加载后: [SceneA, SceneB]

特点:
├─> 保留已加载场景
├─> 叠加新场景
├─> 场景间独立存在
└─> 适用于多场景共存
```

**应用场景：**
- 大型开放世界
- 场景流式加载
- UI独立场景
- 后台加载

### 3. LoadSceneMode.UnloadPreviousScene

```
UnloadPreviousScene模式流程:

加载前: [SceneA]
         ↓ LoadScene("SceneB", Additive)
         ↓ SceneManager.UnloadSceneAsync(SceneA)
加载后: [SceneB]

特点:
├─> 加载新场景
├─> 自动卸载旧场景
└─> 手动控制加载和卸载
```

## 异步加载机制

### AsyncOperation 工作原理

```
AsyncOperation流程:

1. 启动加载
   ├─> 创建AsyncOperation对象
   ├─> 初始化加载状态
   └─> 返回操作句柄

2. 加载过程
   ├─> 读取场景文件 (主线程或后台线程)
   ├─> 解析场景数据 (主线程)
   ├─> 反序列化对象 (主线程)
   ├─> 加载资源 (主线程或后台线程)
   └─> 更新进度 (progress)

3. 激活阶段
   ├─> allowSceneActivation = false
   ├─> 场景已加载但未激活
   ├─> allowSceneActivation = true
   └─> 激活场景

4. 完成
   ├─> isDone = true
   ├─> progress = 1.0
   └─> 触发场景加载完成事件
```

### 异步加载代码解析

```csharp
// AsyncOperation内部机制（简化版）
public class AsyncOperation
{
    public bool isDone { get; private set; }
    public float progress { get; private set; }
    public bool allowSceneActivation { get; set; } = true;
    public int priority { get; set; }

    private float m_Progress = 0f;
    private AsyncOperationState m_State;

    public enum AsyncOperationState
    {
        LoadingSceneFile,
        ParsingSceneData,
        DeserializingObjects,
        LoadingResources,
        Activating,
        Completed
    }

    // 每帧更新进度
    private void Update()
    {
        switch (m_State)
        {
            case AsyncOperationState.LoadingSceneFile:
                // 读取场景文件 (0.0 - 0.2)
                m_Progress = Mathf.Lerp(m_Progress, 0.2f, 0.1f);
                if (m_Progress >= 0.19f)
                    m_State = AsyncOperationState.ParsingSceneData;
                break;

            case AsyncOperationState.ParsingSceneData:
                // 解析场景数据 (0.2 - 0.4)
                m_Progress = Mathf.Lerp(m_Progress, 0.4f, 0.1f);
                if (m_Progress >= 0.39f)
                    m_State = AsyncOperationState.DeserializingObjects;
                break;

            case AsyncOperationState.DeserializingObjects:
                // 反序列化对象 (0.4 - 0.7)
                m_Progress = Mathf.Lerp(m_Progress, 0.7f, 0.1f);
                if (m_Progress >= 0.69f)
                    m_State = AsyncOperationState.LoadingResources;
                break;

            case AsyncOperationState.LoadingResources:
                // 加载资源 (0.7 - 0.9)
                m_Progress = Mathf.Lerp(m_Progress, 0.9f, 0.1f);
                if (m_Progress >= 0.89f)
                    m_State = AsyncOperationState.Activating;
                break;

            case AsyncOperationState.Activating:
                // 激活场景 (0.9 - 1.0)
                if (allowSceneActivation)
                {
                    m_Progress = Mathf.Lerp(m_Progress, 1.0f, 0.1f);
                    if (m_Progress >= 0.99f)
                        m_State = AsyncOperationState.Completed;
                }
                break;

            case AsyncOperationState.Completed:
                m_Progress = 1.0f;
                isDone = true;
                break;
        }

        // 更新公开属性
        progress = m_Progress;
    }
}
```

## 场景加载性能优化

### 优化策略

#### 1. 场景文件优化

```
场景文件大小优化:

1. 减少游戏对象数量
   ├─> 合并静态对象
   ├─> 使用对象池
   └─> 减少不必要的组件

2. 优化纹理和模型
   ├─> 使用压缩纹理格式
   ├─> 优化模型面数
   └─> 使用图集

3. 避免大资源引用
   ├─> 分场景管理资源
   ├─> 使用Addressables
   └─> 延迟加载大资源
```

#### 2. 异步加载优化

```csharp
// 场景预加载管理器
public class ScenePreloader : MonoBehaviour
{
    [SerializeField] private string[] scenesToPreload;
    private Dictionary<string, AsyncOperation> preloadOperations = new();

    public void PreloadScenes()
    {
        foreach (var scene in scenesToPreload)
        {
            // 预加载场景（不激活）
            StartCoroutine(PreloadSceneAsync(scene));
        }
    }

    private IEnumerator PreloadSceneAsync(string sceneName)
    {
        // Additive模式加载场景
        var asyncLoad = SceneManager.LoadSceneAsync(sceneName, LoadSceneMode.Additive);
        
        // 阻止自动激活
        asyncLoad.allowSceneActivation = false;
        
        // 存储加载操作
        preloadOperations[sceneName] = asyncLoad;
        
        // 等待加载完成（但不激活）
        while (!asyncLoad.isDone)
        {
            yield return null;
        }
        
        Debug.Log($"Scene {sceneName} preloaded");
    }

    public void ActivateScene(string sceneName)
    {
        if (preloadOperations.TryGetValue(sceneName, out var asyncLoad))
        {
            // 激活场景
            asyncLoad.allowSceneActivation = true;
            preloadOperations.Remove(sceneName);
        }
    }
}
```

#### 3. 场景卸载优化

```csharp
// 场景卸载管理器
public class SceneUnloader : MonoBehaviour
{
    public void UnloadScene(Scene scene)
    {
        StartCoroutine(UnloadSceneAsync(scene));
    }

    private IEnumerator UnloadSceneAsync(Scene scene)
    {
        // 异步卸载场景
        var asyncUnload = SceneManager.UnloadSceneAsync(scene);

        // 等待卸载完成
        while (!asyncUnload.isDone)
        {
            yield return null;
        }

        Debug.Log($"Scene {scene.name} unloaded");
    }

    // 卸载所有Additive场景
    public void UnloadAllAdditiveScenes()
    {
        for (int i = 0; i < SceneManager.sceneCount; i++)
        {
            var scene = SceneManager.GetSceneAt(i);
            if (scene != SceneManager.GetActiveScene() && scene.isLoaded)
            {
                UnloadScene(scene);
            }
        }
    }
}
```

## 场景合并机制

### Additive场景合并原理

```
场景合并流程:

1. 加载基础场景 [BaseScene]
         ↓
2. Additive加载场景A [BaseScene + SceneA]
         ↓
3. Additive加载场景B [BaseScene + SceneA + SceneB]
         ↓
4. 场景合并

合并规则:
├─> 不同场景的对象保持独立
├─> 跨场景引用需要特殊处理
├─> 灯光、相机等全局组件需要管理
└─> 物理系统需要配置
```

### 跨场景通信

```csharp
// 场景管理器 - 跨场景通信
public class SceneManager : MonoBehaviour
{
    private static SceneManager instance;
    public static SceneManager Instance => instance;

    // 场景数据容器
    public Dictionary<string, object> sceneData = new();

    private void Awake()
    {
        if (instance == null)
        {
            instance = this;
            DontDestroyOnLoad(gameObject);
        }
        else
        {
            Destroy(gameObject);
        }
    }

    // 存储场景数据
    public void SetSceneData(string key, object value)
    {
        sceneData[key] = value;
    }

    // 获取场景数据
    public T GetSceneData<T>(string key, T defaultValue = default(T))
    {
        if (sceneData.TryGetValue(key, out var value))
        {
            return (T)value;
        }
        return defaultValue;
    }

    // 场景切换时传递数据
    public void LoadSceneWithData(string sceneName, Dictionary<string, object> data)
    {
        // 存储数据
        foreach (var kvp in data)
        {
            sceneData[kvp.Key] = kvp.Value;
        }

        // 加载场景
        UnityEngine.SceneManagement.SceneManager.LoadScene(sceneName);
    }
}

// 在新场景中使用数据
public class NewSceneController : MonoBehaviour
{
    private void Start()
    {
        // 从SceneManager获取数据
        var playerLevel = SceneManager.Instance.GetSceneData<int>("PlayerLevel", 1);
        var playerHP = SceneManager.Instance.GetSceneData<float>("PlayerHP", 100f);

        Debug.Log($"Player Level: {playerLevel}, HP: {playerHP}");
    }
}
```

## 场景加载事件

### 场景加载生命周期事件

```csharp
// 场景事件监听器
public class SceneEventListener : MonoBehaviour
{
    private void OnEnable()
    {
        // 订阅场景加载事件
        SceneManager.sceneLoaded += OnSceneLoaded;
        SceneManager.sceneUnloaded += OnSceneUnloaded;
        SceneManager.activeSceneChanged += OnActiveSceneChanged;
    }

    private void OnDisable()
    {
        // 取消订阅
        SceneManager.sceneLoaded -= OnSceneLoaded;
        SceneManager.sceneUnloaded -= OnSceneUnloaded;
        SceneManager.activeSceneChanged -= OnActiveSceneChanged;
    }

    // 场景加载完成
    private void OnSceneLoaded(Scene scene, LoadSceneMode mode)
    {
        Debug.Log($"Scene loaded: {scene.name}, Mode: {mode}");

        // 初始化场景
        InitializeScene(scene);
    }

    // 场景卸载完成
    private void OnSceneUnloaded(Scene scene)
    {
        Debug.Log($"Scene unloaded: {scene.name}");

        // 清理场景资源
        CleanupScene(scene);
    }

    // 活跃场景改变
    private void OnActiveSceneChanged(Scene prevScene, Scene newScene)
    {
        Debug.Log($"Active scene changed: {prevScene.name} -> {newScene.name}");
    }

    private void InitializeScene(Scene scene)
    {
        // 初始化场景逻辑
        var rootObjects = scene.GetRootGameObjects();
        foreach (var root in rootObjects)
        {
            // 初始化根对象
            InitializeGameObject(root);
        }
    }

    private void CleanupScene(Scene scene)
    {
        // 清理场景资源
        var rootObjects = scene.GetRootGameObjects();
        foreach (var root in rootObjects)
        {
            // 清理对象
            CleanupGameObject(root);
        }
    }

    private void InitializeGameObject(GameObject obj)
    {
        // 初始化游戏对象
        Debug.Log($"Initializing: {obj.name}");
    }

    private void CleanupGameObject(GameObject obj)
    {
        // 清理游戏对象
        Debug.Log($"Cleaning up: {obj.name}");
    }
}
```

## 最佳实践

### DO ✅

- 使用异步加载场景
- 使用Additive模式实现流式加载
- 预加载下一个场景
- 合理管理场景资源
- 使用SceneManager事件
- 异步卸载场景
- 跨场景使用DontDestroyOnLoad
- 优化场景文件大小

### DON'T ❌

- 不要在主线程同步加载大场景
- 不要忘记卸载不用的场景
- 不要混合使用Sync和Async API
- 不要忽略场景加载错误
- 不要在场景切换时阻塞主线程
- 不要过度使用Additive场景
- 不要忘记处理场景事件
- 不要在场景间传递复杂数据
