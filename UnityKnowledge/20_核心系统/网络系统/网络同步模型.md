# 网络同步模型

> 第2课 | 网络编程实战模块

## 1. 同步模型概述

### 1.1 三种主流模型

| 模型 | 特点 | 适用场景 |
|------|------|----------|
| **状态同步** | 同步游戏状态 | MMO、FPS |
| **帧同步** | 同步玩家输入 | MOBA、RTS |
| **预测回滚** | 本地预测+服务器校正 | 格斗、FPS |

## 2. 状态同步

### 2.1 原理

```
客户端 A                服务器                客户端 B
   │                      │                      │
   ├─── 发送位置 ────────→│                      │
   │                      ├─── 广播位置 ─────────→
   │                      │                      │
   │                      │←──── 发送位置 ────────┤
   │←──── 广播位置 ───────┤                      │
```

### 2.2 实现

```csharp
using System.Collections.Generic;
using UnityEngine;

public class StateSyncClient : MonoBehaviour
{
    private float syncInterval = 0.05f;  // 20次/秒
    private float lastSyncTime;
    private Vector3 lastSyncPosition;
    private NetworkClient networkClient;

    // 状态快照
    private struct StateSnapshot
    {
        public float timestamp;
        public Vector3 position;
        public Quaternion rotation;
    }

    private Queue<StateSnapshot> serverStates = new Queue<StateSnapshot>();

    private void Update()
    {
        // 发送本地状态
        if (IsLocalPlayer() && Time.time - lastSyncTime >= syncInterval)
        {
            SendStateToServer();
            lastSyncTime = Time.time;
        }

        // 插值显示其他玩家
        if (!IsLocalPlayer())
        {
            InterpolatePosition();
        }
    }

    private void SendStateToServer()
    {
        Vector3 position = transform.position;
        Quaternion rotation = transform.rotation;

        // 只有位置变化才发送
        if (Vector3.Distance(position, lastSyncPosition) > 0.01f)
        {
            var packet = new PositionPacket
            {
                x = position.x,
                y = position.y,
                z = position.z,
                rotX = rotation.eulerAngles.x,
                rotY = rotation.eulerAngles.y,
                rotZ = rotation.eulerAngles.z,
                timestamp = Time.time
            };

            networkClient.Send(packet);
            lastSyncPosition = position;
        }
    }

    // 接收服务器状态
    public void OnServerStateReceived(StateSnapshot snapshot)
    {
        serverStates.Enqueue(snapshot);

        // 限制缓冲区大小
        while (serverStates.Count > 10)
        {
            serverStates.Dequeue();
        }
    }

    // 插值平滑显示
    private float interpolationDelay = 0.1f;  // 100ms 延迟

    private void InterpolatePosition()
    {
        if (serverStates.Count < 2) return;

        float renderTime = Time.time - interpolationDelay;

        // 找到两个快照进行插值
        StateSnapshot[] snapshots = serverStates.ToArray();

        for (int i = 0; i < snapshots.Length - 1; i++)
        {
            if (snapshots[i].timestamp <= renderTime && snapshots[i + 1].timestamp >= renderTime)
            {
                float t = (renderTime - snapshots[i].timestamp) /
                          (snapshots[i + 1].timestamp - snapshots[i].timestamp);

                transform.position = Vector3.Lerp(snapshots[i].position, snapshots[i + 1].position, t);
                break;
            }
        }
    }

    private bool IsLocalPlayer() => true;
}
```

## 3. 帧同步

### 3.1 原理

```
回合 1: 所有客户端发送输入 → 服务器收集 → 广播输入 → 所有客户端执行
回合 2: 所有客户端发送输入 → 服务器收集 → 广播输入 → 所有客户端执行
...
```

**关键点**：
- 所有客户端**确定性执行**
- 只同步**输入**，不同步状态
- 每回合等待所有玩家输入

### 3.2 实现

```csharp
using System.Collections.Generic;

public class FrameSyncManager : MonoBehaviour
{
    private int currentFrame = 0;
    private int framesPerTurn = 5;  // 每回合 5 帧
    private float fixedDeltaTime = 1f / 60f;  // 60 FPS

    // 输入缓冲
    private Dictionary<int, Dictionary<int, PlayerInput>> turnInputs = new();

    // 当前回合的输入
    private Dictionary<int, PlayerInput> currentTurnInputs = new();

    private NetworkClient networkClient;
    private int localPlayerId;

    private void FixedUpdate()
    {
        currentFrame++;

        // 收集本地输入
        var localInput = CollectLocalInput();
        currentTurnInputs[localPlayerId] = localInput;

        // 发送输入到服务器
        networkClient.SendInput(currentFrame, localInput);

        // 回合结束，执行所有输入
        if (currentFrame % framesPerTurn == 0)
        {
            ExecuteTurn();
        }
    }

    private PlayerInput CollectLocalInput()
    {
        return new PlayerInput
        {
            moveX = Input.GetAxis("Horizontal"),
            moveY = Input.GetAxis("Vertical"),
            attack = Input.GetButton("Fire1"),
            skillId = GetSkillInput()
        };
    }

    // 接收服务器广播的输入
    public void OnTurnInputsReceived(int turn, Dictionary<int, PlayerInput> inputs)
    {
        turnInputs[turn] = inputs;
    }

    private void ExecuteTurn()
    {
        int turn = currentFrame / framesPerTurn;

        if (!turnInputs.TryGetValue(turn, out var inputs))
        {
            // 等待服务器输入
            return;
        }

        // 确定性执行所有玩家输入
        foreach (var kvp in inputs)
        {
            int playerId = kvp.Key;
            PlayerInput input = kvp.Value;

            ExecutePlayerInput(playerId, input);
        }
    }

    private void ExecutePlayerInput(int playerId, PlayerInput input)
    {
        // 获取玩家实体
        var player = GetPlayerEntity(playerId);

        // 确定性执行
        player.Move(new Vector3(input.moveX, 0, input.moveY));

        if (input.attack)
        {
            player.Attack();
        }

        if (input.skillId > 0)
        {
            player.UseSkill(input.skillId);
        }
    }

    private object GetPlayerEntity(int playerId) => null;
}

public struct PlayerInput
{
    public float moveX;
    public float moveY;
    public bool attack;
    public int skillId;
}
```

```csharp
void BadUpdate()
{
    if (Random.value > 0.5f)  // 每次不同！
    {
        Attack();
    }
}

// ✅ 正确：使用确定性随机
public class DeterministicRandom
{
    private uint seed;

    public DeterministicRandom(uint seed)
    {
        this.seed = seed;
    }

    public int NextInt()
    {
        // Xorshift 算法
        seed ^= seed << 13;
        seed ^= seed >> 17;
        seed ^= seed << 5;
        return (int)(seed & 0x7FFFFFFF);
    }

    public float NextFloat()
    {
        return NextInt() / (float)int.MaxValue;
    }
}

// 使用同步的种子
void GoodUpdate(int turnSeed)
{
    var random = new DeterministicRandom((uint)turnSeed);
    if (random.NextFloat() > 0.5f)  // 所有客户端结果相同
    {
        Attack();
    }
}
```

## 4. 预测与回滚

### 4.1 原理

```
1. 客户端预测：立即执行本地输入
2. 服务器确认：收到服务器状态后校验
3. 回滚修正：如果预测错误，回滚并重新模拟
```

### 4.2 客户端预测实现

```csharp
public class PredictionManager : MonoBehaviour
{
    private struct PredictedState
    {
        public int inputSequence;
        public Vector3 position;
        public Quaternion rotation;
    }

    private Queue<PredictedState> predictedStates = new Queue<PredictedState>();
    private int inputSequence = 0;

    private Vector3 serverPosition;
    private float reconciliationThreshold = 2f;  // 误差阈值

    public void ProcessLocalInput(Vector3 input)
    {
        // 本地预测执行
        PredictMovement(input);

        // 保存预测状态
        predictedStates.Enqueue(new PredictedState
        {
            inputSequence = inputSequence,
            position = transform.position,
            rotation = transform.rotation
        });

        // 发送输入到服务器
        SendInputToServer(inputSequence, input);
        inputSequence++;
    }

    private void PredictMovement(Vector3 input)
    {
        // 立即执行移动
        transform.position += input * Time.deltaTime * 5f;
    }

    // 接收服务器状态
    public void OnServerStateReceived(int sequence, Vector3 serverPos)
    {
        serverPosition = serverPos;

        // 移除已确认的预测
        while (predictedStates.Count > 0 && predictedStates.Peek().inputSequence <= sequence)
        {
            var confirmed = predictedStates.Dequeue();

            // 检查误差
            float error = Vector3.Distance(confirmed.position, serverPos);

            if (error > reconciliationThreshold)
            {
                // 预测错误，需要回滚
                Reconcile(serverPos);
            }
        }
    }

    private void Reconcile(Vector3 correctPosition)
    {
        // 回滚到正确位置
        transform.position = correctPosition;

        // 重新模拟所有未确认的输入
```

## 5. 延迟补偿

### 5.1 服务器端延迟补偿

```csharp
public class LagCompensation
{
    private struct PlayerHistory
    {
        public float timestamp;
        public Vector3 position;
        public Quaternion rotation;
    }

    private Dictionary<int, List<PlayerHistory>> playerHistories = new();
    private float historyDuration = 1f;  // 保存 1 秒历史

    // 记录玩家位置历史
    public void RecordPlayerPosition(int playerId, Vector3 position, Quaternion rotation)
    {
        if (!playerHistories.ContainsKey(playerId))
            playerHistories[playerId] = new List<PlayerHistory>();

        playerHistories[playerId].Add(new PlayerHistory
        {
            timestamp = Time.time,
            position = position,
            rotation = rotation
        });

        // 清理过期历史
        var history = playerHistories[playerId];
        history.RemoveAll(h => Time.time - h.timestamp > historyDuration);
    }

    // 获取指定时间点的玩家位置
    public Vector3 GetPlayerPositionAtTime(int playerId, float targetTime)
    {
        var history = playerHistories[playerId];

        for (int i = 0; i < history.Count - 1; i++)
        {
            if (history[i].timestamp <= targetTime && history[i + 1].timestamp >= targetTime)
            {
                float t = (targetTime - history[i].timestamp) /
                          (history[i + 1].timestamp - history[i].timestamp);
                return Vector3.Lerp(history[i].position, history[i + 1].position, t);
            }
        }

        return history[history.Count - 1].position;
    }

    // 处理射击（带延迟补偿）
    public void ProcessShot(int shooterId, Vector3 origin, Vector3 direction, float shooterLatency)
    {
        // 计算射击时的服务器时间
        float shotTime = Time.time - shooterLatency;

        // 检测命中（使用历史位置）
        foreach (var playerId in playerHistories.Keys)
        {
            if (playerId == shooterId) continue;

            Vector3 historicalPosition = GetPlayerPositionAtTime(playerId, shotTime);

            // 射线检测
            if (Physics.Raycast(origin, direction, out var hit, 100f))
            {
                if (Vector3.Distance(hit.point, historicalPosition) < 1f)
                {
                    // 命中！
                    ApplyDamage(playerId, 10);
                }
            }
        }
    }
```

## 本课小结

### 同步模型对比

| 特性 | 状态同步 | 帧同步 | 预测回滚 |
|------|----------|--------|----------|
| 带宽 | 高 | 低 | 中 |
| 响应速度 | 慢 | 快 | 最快 |
| 实现难度 | 低 | 高 | 高 |
| 作弊防护 | 好 | 差 | 中 |
| 适用 | MMO/FPS | MOBA/RTS | 格斗/FPS |

### 关键技术

| 技术 | 用途 |
|------|------|
| 插值 | 平滑显示其他玩家 |
| 外推 | 预测未来位置 |
| 预测 | 本地立即响应 |
| 回滚 | 修正预测错误 |
| 延迟补偿 | 公平性保证 |
