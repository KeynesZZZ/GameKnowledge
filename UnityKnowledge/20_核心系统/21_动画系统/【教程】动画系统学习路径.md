## 核心知识点

### 1. Animator 状态机

| 概念 | 说明 |
|------|------|
| State | 动画状态 |
| Transition | 状态过渡 |
| Parameter | 控制参数 |
| Layer | 动画层 |
| Avatar Mask | 身体遮罩 |

### 2. 混合树

| 类型 | 用途 |
|------|------|
| 1D | 单参数混合（速度） |
| 2D Simple Directional | 方向+速度 |
| 2D Freeform Directional | 自由方向 |
| Direct | 权重直接控制 |

### 3. IK 系统

```csharp
void OnAnimatorIK(int layerIndex)
{
    animator.SetIKPosition(AvatarIKGoal.RightHand, targetPos);
    animator.SetIKPositionWeight(AvatarIKGoal.RightHand, 1f);
}
```

### 4. Playables API

| 类 | 用途 |
|------|------|
| PlayableGraph | 播放图 |
| AnimationMixerPlayable | 动画混合器 |
| AnimationClipPlayable | 动画片段 |
| AnimationLayerMixerPlayable | 层混合器 |
