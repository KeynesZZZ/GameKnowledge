## 概述

Blend Tree 是 Unity Animator 的强大功能，允许在多个动画之间进行平滑混合。它对于创建自然、流畅的角色动画至关重要。

## Blend Tree 类型

### 1. 1D Blend Tree

**基于单个参数（如速度）**

```csharp
using UnityEngine;

public class BlendTree1D : MonoBehaviour
{
    public Animator animator;

    void Update()
    {
        // 获取速度
        float speed = Input.GetAxis("Vertical");
        
        // 设置 Blend Tree 参数
        animator.SetFloat("Speed", Mathf.Abs(speed));
        
        // 触发动画状态
        animator.SetBool("IsMoving", Mathf.Abs(speed) > 0.1f);
    }
}
```

### 2. 2D Blend Tree

**基于两个参数（如 X 和 Z 移动）**

```csharp
public class BlendTree2D : MonoBehaviour
{
    public Animator animator;

    void Update()
    {
        // 获取移动输入
        float horizontal = Input.GetAxis("Horizontal");
        float vertical = Input.GetAxis("Vertical");
        
        // 设置 2D Blend Tree 参数
        animator.SetFloat("MoveX", horizontal);
        animator.SetFloat("MoveZ", vertical);
        
        // 触发移动状态
        animator.SetBool("IsMoving", 
            Mathf.Abs(horizontal) > 0.1f || 
            Mathf.Abs(vertical) > 0.1f
        );
    }
}
```

### 3. Direct Blend Tree

**直接控制混合权重**

```csharp
public class DirectBlend : MonoBehaviour
{
    public Animator animator;

    void Update()
    {
        // 直接设置混合权重
        animator.SetFloat("UpperBodyWeight", 0.7f);
        animator.SetFloat("LowerBodyWeight", 0.3f);
    }
}
```

## 过渡条件

### 过渡条件配置

在 Animator Controller 中配置过渡条件：

#### 1. 基于触发器

```
参数类型: Trigger
条件: When true
```

#### 2. 基于布尔值

```
参数类型: Bool
条件: When true (进入状态) / When false (退出状态)
```

#### 3. 基于浮点值

```
参数类型: Float
条件: Greater (大于) / Less (小于)
```

### 过渡时间控制

```csharp
public class TransitionControl : MonoBehaviour
{
    public Animator animator;

    void Start()
    {
        // 设置全局过渡时间
        // 也可以在 Inspector 中设置
    }

    public void PlayAnimation(string stateName, float transitionDuration = 0.2f)
    {
        animator.CrossFade(stateName, transitionDuration);
    }

    public void PlayInstant(string stateName)
    {
        animator.Play(stateName, 0, 0f);  // 无过渡
    }
}
```

## 实战示例

### 1. 角色移动 Blend Tree

```csharp
using UnityEngine;

[RequireComponent(typeof(Animator))]
public class CharacterMovementBlend : MonoBehaviour
{
    public float walkSpeed = 2f;
    public float runSpeed = 5f;

    private Animator animator;

    void Start()
    {
        animator = GetComponent<Animator>();
    }

    void Update()
    {
        // 计算移动速度
        float horizontal = Input.GetAxis("Horizontal");
        float vertical = Input.GetAxis("Vertical");
        
        Vector3 movement = new Vector3(horizontal, 0f, vertical);
        float speed = movement.magnitude * (Input.GetButton("Sprint") ? runSpeed : walkSpeed);
        
        // 设置 Blend Tree 参数
        animator.SetFloat("Speed", speed);
        animator.SetFloat("MoveX", horizontal);
        animator.SetFloat("MoveZ", vertical);
        
        // 触发状态
        animator.SetBool("IsMoving", speed > 0.1f);
        animator.SetBool("IsRunning", Input.GetButton("Sprint"));
    }
}
```

### 2. 攻击动画过渡

```csharp
public class AttackTransition : MonoBehaviour
{
    public float attackTransitionTime = 0.1f;
    public float attackDuration = 0.5f;

    private Animator animator;
    private bool isAttacking;

    void Start()
    {
        animator = GetComponent<Animator>();
    }

    void Update()
    {
        if (Input.GetButtonDown("Attack") && !isAttacking)
        {
            StartAttack();
        }
    }

    void StartAttack()
    {
        isAttacking = true;
        
        // 平滑过渡到攻击动画
        animator.CrossFade("Attack", attackTransitionTime);
        
        // 延时恢复
        Invoke("EndAttack", attackDuration);
    }

    void EndAttack()
    {
        isAttacking = false;
        
        // 平滑过渡回待机
        animator.CrossFade("Idle", attackTransitionTime);
    }
}
```

### 3. 上下半身分离

```csharp
[RequireComponent(typeof(Animator))]
public class UpperLowerBodyBlend : MonoBehaviour
{
    public float upperBodyLayer = 0;
    public float lowerBodyLayer = 1;

    private Animator animator;

    void Start()
    {
        animator = GetComponent<Animator>();
        
        // 设置层级权重
        animator.SetLayerWeight(upperBodyLayer, 1.0f);
        animator.SetLayerWeight(lowerBodyLayer, 1.0f);
    }

    void Update()
    {
        // 上半身（手臂）动画
        float attack = Input.GetAxis("Attack");
        animator.SetFloat("UpperBodyWeight", attack);
        
        // 下半身（腿部）动画
        float move = Input.GetAxis("Vertical");
        animator.SetFloat("LowerBodyWeight", move);
    }
}
```

## 最佳实践

### DO ✅

- 使用 Blend Tree 创建流畅的动画
- 为 Blend Tree 设置合理的参数范围
- 使用过渡时间控制动画流畅度
- 使用 CrossFade 而不是 Play（更平滑）
- 合理设置过渡条件和时间
- 使用动画层分离上下半身

### DON'T ❌

- 不要使用过多的 Blend Tree（性能问题）
- 不要设置过短的过渡时间（突兀）
- 不要忘记设置 Blend Tree 参数范围
- 不要混淆触发器和布尔值的使用场景
- 不要在 Update 中频繁调用 CrossFade

## 常见问题

### Q: 动画过渡不自然？
**A**: 
1. 增加过渡时间
2. 检查 Blend Tree 参数是否正确
3. 确保动画有合适的根运动
4. 使用 CrossFade 而不是 Play

### Q: Blend Tree 不工作？
**A**: 
1. 检查参数名称是否匹配
2. 检查参数类型是否正确
3. 检查 Blend Tree 是否正确设置
4. 检查动画剪辑是否正确导入
