## 概述

后处理在场景渲染完成后对最终图像进行效果处理，如 Bloom、DOF、色差等。Unity 提供了 Post-Processing Stack 简化后处理效果实现。

## Post-Processing Stack 基础

### 1. 安装和配置

```
Window > Package Manager
搜索: Post Processing
安装: Post Processing Stack v2
```

### 2. 基础配置

```csharp
using UnityEngine;
using UnityEngine.Rendering.PostProcessing;

public class PostProcessingSetup : MonoBehaviour
{
    public PostProcessVolume volume;
    
    private Bloom bloom;
    private ColorGrading colorGrading;
    private DepthOfField depthOfField;

    void Start()
    {
        // 获取效果组件
        bloom = volume.profile.GetSetting<Bloom>();
        colorGrading = volume.profile.GetSetting<ColorGrading>();
        depthOfField = volume.profile.GetSetting<DepthOfField>();
        
        // 配置效果
        ConfigureEffects();
    }

    void ConfigureEffects()
    {
        // Bloom 效果
        bloom.intensity.value = 1.0f;
        bloom.threshold.value = 1.0f;
        bloom.radius.value = 4.0f;
        
        // 色差效果
        colorGrading.enabled.value = true;
        
        // DOF 效果
        depthOfField.focusDistance.value = 10.0f;
        depthOfField.aperture.value = 5.6f;
    }
}
```

## 实战案例

### 1. Bloom 发光效果

```csharp
using UnityEngine.Rendering.PostProcessing;

public class BloomEffect : MonoBehaviour
{
    public PostProcessVolume volume;
    
    private Bloom bloom;

    void Start()
    {
        bloom = volume.profile.GetSetting<Bloom>();
        
        // 动态调整 Bloom
        AnimateBloom();
    }

    void AnimateBloom()
    {
        LeanTween.value(
            bloom.threshold.value,
            1.5f,
            2.0f
        ).setEase(LeanTweenType.easeInOutQuad);
    }
}
```

### 2. 运动模糊

```csharp
using UnityEngine.Rendering.PostProcessing;

public class MotionBlur : MonoBehaviour
{
    public PostProcessVolume volume;
    
    private MotionBlur motionBlur;

    void Update()
    {
        motionBlur = volume.profile.GetSetting<MotionBlur>();
        
        // 根据速度调整模糊
        float speed = rigidbody.velocity.magnitude;
        motionBlur.shutterAngle.value = Mathf.Clamp(speed * 10f, 0f, 270f);
    }
}
```

### 3. 色差效果

```csharp
using UnityEngine.Rendering.PostProcessing;

public class ChromaticAberration : MonoBehaviour
{
    public PostProcessVolume volume;
    
    private ChromaticAberration chromaticAberration;

    void Start()
    {
        chromaticAberration = volume.profile.GetSetting<ChromaticAberration>();
        
        // 模拟镜头色差
        chromaticAberration.intensity.value = 0.1f;
        chromaticAberration.spectralLut.value = null;
    }
}
```

## 性能优化

### 1. 效果分级

```csharp
public class PerformanceSettings : MonoBehaviour
{
    public int qualityLevel = 2; // 0=低, 1=中, 2=高
    
    void SetQuality()
    {
        switch (qualityLevel)
        {
            case 0: // 低质量
                DisableExpensiveEffects();
                break;
            case 1: // 中质量
                EnableMediumEffects();
                break;
            case 2: // 高质量
                EnableAllEffects();
                break;
        }
    }
    
    void DisableExpensiveEffects()
    {
        // 禁用昂贵的后处理效果
        // Bloom: 关闭
        // DOF: 关闭
    }
}
```

### 2. 减少后处理分辨率

```csharp
public class Downsampling : MonoBehaviour
{
    public PostProcessLayer layer;
    
    void Start()
    {
        // 降低后处理分辨率以提升性能
        layer.downscaling.value = 2; // 2x 降采样
    }
}
```

## 最佳实践

### DO ✅

- 使用 Post-Processing Stack 简化后处理
- 根据平台调整效果强度
- 使用分级质量设置
- 在低端设备禁用昂贵效果
- 使用降采样提升性能

### DON'T ❌

- 不要同时启用过多的后处理效果
- 不要在移动端使用高质量后处理
- 不要忽略后处理的性能成本
- 不要在 Update 中频繁调整效果参数

## 常见问题

### Q: 后处理不生效？
**A**: 
1. 检查 Post-Processing Stack 是否正确安装
2. 检查 Volume 是否正确配置
3. 检查 Camera 是否添加了 Layer
4. 检查效果是否启用

### Q: 后处理性能太差？
**A**: 
1. 禁用部分效果
2. 降低分辨率
3. 使用质量分级
4. 使用降采样
