# 2D 物理系统专项

> Unity 2D 物理系统完整指南，与 3D 物理的对比与优化 `#物理系统` `#2D物理` `#性能优化`

## 概述

Unity 2D 物理系统与 3D 物理系统有显著差异。2D 物理更适合 2D 游戏，提供更好的性能和准确的行为。

## 2D vs 3D 物理对比

### 核心组件对比

| 功能 | 3D 物理 | 2D 物理 |
|------|---------|---------|
| **刚体** | Rigidbody | Rigidbody2D |
| **碰撞体** | Collider | Collider2D |
| **关节** | Joint | Joint2D |
| **触发器** | OnTriggerEnter | OnTriggerEnter2D |
| **坐标** | Vector3 (x, y, z) | Vector2 (x, y) |
| **旋转** | Quaternion | float (角度) |

### 配置对比


## Rigidbody2D 配置

### 类型选择

```csharp
public enum BodyType2D
{
    Dynamic,    // 动态物体（受物理影响）
    Kinematic,   // 运动学物体（不受物理影响，可控制位置）
    Static      // 静态物体（不可移动）
}
```

| 类型 | 受物理力 | 可控制位置 | 使用场景 |
|------|----------|-----------|----------|
| **Dynamic** | ✅ 是 | ⚠️ 有限 | 角色、敌人、可交互物体 |
| **Kinematic** | ❌ 否 | ✅ 是 | 平台、传送门、触发器 |
| **Static** | ❌ 否 | ❌ 否 | 地面、墙壁、静态场景 |

### 2D 特定配置

```csharp
public class Rigidbody2DConfig : MonoBehaviour
{
    public Rigidbody2D rb2d;

    void Start()
    {
        // 2D 特有配置
        rb2d.gravityScale = 1.0f;      // 重力缩放
        rb2d.collisionDetection = CollisionDetectionMode2D.Continuous;
        rb2d.sleepMode = RigidbodySleepMode2D.StartAwake;
        
        // 模拟约束（3D 方式）
        rb2d.constraints = RigidbodyConstraints2D.FreezeRotation;
    }
}
```

## 2D 关节

### 常用关节类型

```csharp
using UnityEngine;

public class Joint2DExamples : MonoBehaviour
{
    // 1. Spring Joint（弹簧关节）
    public void CreateSpringJoint()
    {
        var joint = gameObject.AddComponent<SpringJoint2D>();
        joint.connectedBody = otherRigidbody2D;
        joint.frequency = 5f;
        joint.dampingRatio = 0.7f;
    }

    // 2. Distance Joint（距离关节）
    public void CreateDistanceJoint()
    {
        var joint = gameObject.AddComponent<DistanceJoint2D>();
        joint.connectedBody = otherRigidbody2D;
        joint.maxDistanceOnly = true;
        joint.distance = 2f;
    }

    // 3. Hinge Joint（铰链关节）
    public void CreateHingeJoint()
    {
        var joint = gameObject.AddComponent<HingeJoint2D>();
        joint.connectedBody = otherRigidbody2D;
        joint.useLimits = true;
        joint.limits = new JointAngleLimits2D
        {
            min = -90f,
            max = 90f
        };
    }

    // 4. Slider Joint（滑动关节）
    public void CreateSliderJoint()
    {
        var joint = gameObject.AddComponent<SliderJoint2D>();
        joint.connectedBody = otherRigidbody2D;
        joint.useLimits = true;
        joint.limits = new JointTranslationLimits2D
        {
            min = -2f,
            max = 2f
        };
    }
}
```

| 关节 | 用途 | 性能 |
|------|------|------|
| **Spring** | 弹簧、弹性连接 | ⭐⭐⭐ |
| **Distance** | 固定距离、绳索 | ⭐⭐⭐⭐ |
| **Hinge** | 旋转约束 | ⭐⭐⭐⭐ |
| **Slider** | 滑动约束 | ⭐⭐⭐⭐ |

## 性能优化

### 1. 使用合适的碰撞体

```csharp
// ❌ 错误：使用 Polygon Collider（慢）
var polygon = gameObject.AddComponent<PolygonCollider2D>();

// ✅ 正确：使用 Box/Circle Collider（快）

```

### 2. 减少不必要的物理计算

```csharp
public class OptimizedPhysics2D : MonoBehaviour
{
    private Rigidbody2D rb2d;
    private bool isKinematic = false;

    void Start()
    {
        rb2d = GetComponent<Rigidbody2D>();
    }

    void Update()
    {
        // 远离玩家时切换为 Kinematic
        float distance = Vector2.Distance(transform.position, player.position);
        if (distance > 20f && !isKinematic)
        {
            rb2d.isKinematic = true;
            isKinematic = true;
        }
        else if (distance < 15f && isKinematic)
        {
            rb2d.isKinematic = false;
            isKinematic = false;
        }
    }
}
```

## 实战示例

### 1. 2D 角色控制器

```csharp
using UnityEngine;

public class CharacterController2D : MonoBehaviour
{
    public float moveSpeed = 5f;
    public float jumpForce = 8f;
    public float groundDistance = 0.1f;
    public LayerMask groundLayer;

    private Rigidbody2D rb2d;
    private bool isGrounded;

    void Start()
    {
        rb2d = GetComponent<Rigidbody2D>();
        
        // 配置 Rigidbody2D
        rb2d.mass = 1.0f;
        rb2d.drag = 0.5f;
        rb2d.gravityScale = 1.0f;
        rb2d.constraints = RigidbodyConstraints2D.FreezeRotation;
        rb2d.collisionDetection = CollisionDetectionMode2D.Continuous;
    }

    void Update()
    {
        CheckGround();
        HandleInput();
    }

    void CheckGround()
    {
        isGrounded = Physics2D.Raycast(
            transform.position + Vector2.down * 0.05f,
            Vector2.down,
            groundDistance,
            groundLayer
        );
    }

    void HandleInput()
    {
        float horizontal = Input.GetAxis("Horizontal");
        
        // 移动
        rb2d.velocity = new Vector2(horizontal * moveSpeed, rb2d.velocity.y);

        // 跳跃
        if (isGrounded && Input.GetButtonDown("Jump"))
        {
            rb2d.velocity = new Vector2(rb2d.velocity.x, jumpForce);
        }
    }

    void OnDrawGizmos()
    {
        Gizmos.color = isGrounded ? Color.green : Color.red;
        Gizmos.DrawRay(transform.position, Vector2.down * groundDistance);
    }
}
```

### 2. 2D 平台生成

```csharp
public class PlatformSpawner : MonoBehaviour
{
    public GameObject platformPrefab;
    public int platformCount = 10;
    public float spawnRange = 20f;

    void Start()
    {
        for (int i = 0; i < platformCount; i++)
        {
            Vector2 spawnPos = new Vector2(
                Random.Range(-spawnRange, spawnRange),
                Random.Range(-spawnRange, spawnRange)
            );

            GameObject platform = Instantiate(platformPrefab, spawnPos, Quaternion.identity);
            
            // 配置为 Kinematic（静态平台）
            Rigidbody2D rb2d = platform.GetComponent<Rigidbody2D>();
            if (rb2d != null)
            {
                rb2d.isKinematic = true;
            }
        }
    }
}
```

### 4. 2D 关节示例（绳索）

```csharp
public class Rope2D : MonoBehaviour
{
    public GameObject[] ropeSegments;
    public float segmentLength = 0.5f;

    void Start()
    {
        for (int i = 0; i < ropeSegments.Length; i++)
        {
            Rigidbody2D currentSegment = ropeSegments[i].GetComponent<Rigidbody2D>();
            Rigidbody2D previousSegment = null;

            if (i < ropeSegments.Length - 1)
            {
                previousSegment = ropeSegments[i + 1].GetComponent<Rigidbody2D>();
            }

            if (previousSegment != null)
            {
                DistanceJoint2D joint = ropeSegments[i].AddComponent<DistanceJoint2D>();
                joint.connectedBody = previousSegment;
                joint.distance = segmentLength;
                joint.maxDistanceOnly = true;
                joint.dampingRatio = 0.5f;
                joint.frequency = 10f;
            }
        }
    }
}
```

## 最佳实践

### DO ✅

- 2D 游戏使用 Rigidbody2D 和 Collider2D
- 根据形状选择合适的 Collider2D 类型
- 使用 2D 碰撞矩阵优化性能
- 为平台使用 Kinematic Rigidbody2D
- 使用 Joint2D 创建连接
- 使用 Box/Circle Collider2D（比 Polygon 快）

### DON'T ❌

- 不要在 2D 游戏中使用 3D 物理组件
- 不要使用 Polygon Collider2D（除非必要）
- 不要混淆 2D 和 3D 的 API
- 不要忘记配置 2D 碰撞矩阵
- 不要在 Update 中频繁修改 Rigidbody2D
- 不要忽略 Joint2D 的性能影响

## 常见问题

### Q: 2D 和 3D 物理可以混用吗？
**A**: 不可以。2D 和 3D 物理系统是独立的，不能交互。

### Q: 如何在 2D 中模拟 3D 深度？
**A**: 使用 Z 轴或 Layer 系统来管理 2D 中的前后关系。

### Q: Rigidbody2D 性能不够怎么办？
**A**: 
1. 减少 Rigidbody2D 数量
2. 使用合适的 Collider2D 类型
3. 启用睡眠模式
4. 考虑对象池
