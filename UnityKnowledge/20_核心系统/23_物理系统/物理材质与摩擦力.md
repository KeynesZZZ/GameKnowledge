# 物理材质与摩擦力

> Unity 物理材质、摩擦力与弹力的完整配置指南 `#物理系统` `#PhysicsMaterial` `#摩擦力`

## 概述

PhysicsMaterial 控制物体碰撞时的物理行为，包括摩擦力、弹力、摩擦组合模式等。正确配置物理材质对游戏手感至关重要。

## 基础配置

### 创建物理材质

```csharp
// 方法 1：通过代码创建
using UnityEngine;

public class PhysicsMaterialCreator : MonoBehaviour
{
    public PhysicMaterial material;

    void Start()
    {
        material = new PhysicMaterial("GroundMaterial");
        material.dynamicFriction = 0.6f;
        material.staticFriction = 0.6f;
        material.bounciness = 0.0f;
        material.frictionCombine = PhysicMaterialCombine.Average;
    }
}

// 方法 2：通过 Inspector 创建
// Assets > Create > Physic Material
```

### 应用物理材质

```csharp
public class ApplyMaterial : MonoBehaviour
{
    public PhysicMaterial material;
    public Collider collider;

    void Start()
    {
        // 方法 1：应用到 Collider
        collider.material = material;

        // 方法 2：应用到 Rigidbody（全局）
        Rigidbody rb = GetComponent<Rigidbody>();
        if (rb != null)
        {
            // 仅在碰撞时使用此材质
        }
    }
}
```

## 核心参数

### 1. 摩擦力

```csharp
public class FrictionSetup : MonoBehaviour
{
    public PhysicMaterial iceMaterial;
    public PhysicMaterial groundMaterial;

    void Start()
    {
        // 冰面材质（低摩擦）
        iceMaterial.staticFriction = 0.1f;   // 静态摩擦
        iceMaterial.dynamicFriction = 0.05f;  // 动态摩擦

        // 地面材质（正常摩擦）
        groundMaterial.staticFriction = 0.6f;
        groundMaterial.dynamicFriction = 0.5f;
    }
}
```

| 材质 | 静态摩擦 | 动态摩擦 | 描述 |
|------|----------|----------|------|
| **冰面** | 0.1 | 0.05 | 非常滑 |
| **水面** | 0.3 | 0.2 | 滑 |
| **地面** | 0.6 | 0.5 | 正常 |
| **草地** | 0.8 | 0.7 | 抓地力强 |
| **橡胶** | 1.0 | 0.9 | 很强抓地 |

### 3. 摩擦组合模式

```csharp
public class FrictionCombine : MonoBehaviour
{
    public PhysicMaterial material;

    void Start()
    {
        // 平均（推荐）
        material.frictionCombine = PhysicMaterialCombine.Average;

        // 最大值
        material.frictionCombine = PhysicMaterialCombine.Maximum;

        // 最小值
        material.frictionCombine = PhysicMaterialCombine.Multiply;

        // 相乘
        material.frictionCombine = PhysicMaterialCombine.Minimum;
    }
}
```

| 模式 | 效果 | 使用场景 |
|------|------|----------|
| **Average** | 取平均值 | 通用场景 |
| **Maximum** | 取最大值 | 需要强摩擦 |
| **Minimum** | 取最小值 | 需要滑行 |
| **Multiply** | 相乘结果 | 精细控制 |

## 实战示例

### 1. 角色移动

```csharp
using UnityEngine;

public class CharacterMovement : MonoBehaviour
{
    public float moveSpeed = 5f;
    public float groundDistance = 0.2f;
    public LayerMask groundLayer;

    private Rigidbody rb;
    private bool isGrounded;

    void Start()
    {
        rb = GetComponent<Rigidbody>();
        
        // 配置 Rigidbody
        rb.drag = 0.5f;
        rb.angularDrag = 0.5f;
        rb.constraints = RigidbodyConstraints.FreezeRotation;
    }

    void FixedUpdate()
    {
        CheckGround();
        Move();
    }

    void CheckGround()
    {
        isGrounded = Physics.Raycast(
            transform.position + Vector3.down * 0.1f,
            Vector3.down,
            groundDistance,
            groundLayer
        );
    }

    void Move()
    {
        float horizontal = Input.GetAxis("Horizontal");
        float vertical = Input.GetAxis("Vertical");

        Vector3 movement = new Vector3(horizontal, 0f, vertical);

        // 在地面上移动更自然
        if (isGrounded)
        {
            rb.MovePosition(transform.position + movement * moveSpeed * Time.fixedDeltaTime);
        }
        else
        {
            rb.AddForce(movement * moveSpeed * 10f);
        }
    }
}
```

### 2. 车辆物理

```csharp
public class CarPhysics : MonoBehaviour
{
    public float motorTorque = 100f;
    public float brakeTorque = 200f;
    public float maxSteerAngle = 30f;

    public WheelCollider[] wheels;
    public PhysicMaterial tireMaterial;

    void Start()
    {
        // 配置轮胎材质
        tireMaterial.staticFriction = 0.8f;
        tireMaterial.dynamicFriction = 0.6f;
        tireMaterial.bounciness = 0.0f;

        // 应用到轮子
        foreach (WheelCollider wheel in wheels)
        {
            wheel.material = tireMaterial;
            wheel.suspensionDistance = 0.2f;
            wheel.suspensionSpring = new JointSpring
            {
                spring = 35000f,
                damper = 4500f,
                targetPosition = 0.1f
            };
        }
    }

    void FixedUpdate()
    {
        float motor = Input.GetAxis("Vertical");
        float steer = Input.GetAxis("Horizontal");

        // 应用油门
        foreach (WheelCollider wheel in wheels)
        {
            wheel.motorTorque = motor * motorTorque;
            wheel.steerAngle = steer * maxSteerAngle;
            wheel.brakeTorque = 0f;
        }

        // 刹车
        if (Input.GetButton("Brake"))
        {
            foreach (WheelCollider wheel in wheels)
            {
                wheel.brakeTorque = brakeTorque;
                wheel.motorTorque = 0f;
            }
        }
    }
}
```

### 3. 投掷物

```csharp
public class Throwable : MonoBehaviour
{
    public float throwForce = 10f;
    public PhysicMaterial material;

    private Rigidbody rb;

    void Start()
    {
        rb = GetComponent<Rigidbody>();
        rb.material = material;
    }

    public void Throw(Vector3 direction)
    {
        rb.isKinematic = false;
        rb.velocity = direction * throwForce;
        rb.angularVelocity = Vector3.up * 5f;
    }

    void OnCollisionEnter(Collision collision)
    {
        // 碰撞后停止旋转
        rb.angularVelocity = Vector3.zero;
    }
}
```

## 高级技巧

### 1. 动态修改摩擦力

```csharp
public class DynamicFriction : MonoBehaviour
{
    public PhysicMaterial material;
    public float wetFriction = 0.3f;
    public float dryFriction = 0.6f;

    private bool isWet = false;

    void OnTriggerEnter(Collider other)
    {
        if (other.CompareTag("Water"))
        {
            isWet = true;
            UpdateFriction();
        }
    }

    void OnTriggerExit(Collider other)
    {
        if (other.CompareTag("Water"))
        {
            isWet = false;
            UpdateFriction();
        }
    }

    void UpdateFriction()
    {
        material.dynamicFriction = isWet ? wetFriction : dryFriction;
        material.staticFriction = isWet ? wetFriction : dryFriction;
    }
}
```

### 2. 不同材质碰撞

```csharp
public class MaterialCollision : MonoBehaviour
{
    public PhysicMaterial playerMaterial;
    public PhysicMaterial groundMaterial;

    void Start()
    {
        // 玩家与地面的摩擦
        PhysicsMaterialCombine combine = playerMaterial.frictionCombine;

        // 根据碰撞对象调整
        switch (combine)
        {
            case PhysicMaterialCombine.Average:
                // 取平均摩擦
                break;
            case PhysicMaterialCombine.Maximum:
                // 取最大摩擦
                break;
        }
    }
}
```

## 最佳实践

### DO ✅

- 根据游戏类型调整摩擦力和弹力
- 使用 PhysicMaterialCombine 控制摩擦行为
- 为不同表面创建不同的材质
- 测试并调整物理参数
- 在物理材质中添加描述名称

### DON'T ❌

- 不要使用过高的弹力（可能导致物理不稳定）
- 不要忘记为 Collider 分配材质
- 不要混淆静态和动态摩擦
- 不要忽略摩擦组合模式的影响
- 不要在运行时频繁修改物理材质（性能问题）

## 常见问题

### Q: 角色滑行太严重？
**A**: 
1. 增加摩擦力（staticFriction 和 dynamicFriction）
2. 使用 Average 摩擦组合
3. 检查物理材质是否正确应用
4. 考虑使用 Maximum 摩擦组合

### Q: 物体反弹太高？
**A**: 
1. 降低弹力值（bounciness）
2. 添加阻力（drag 和 angularDrag）
3. 使用 Minimum 摩擦组合
4. 考虑使用静态摩擦更高

### Q: 如何让物体完全不滑？
**A**: 
1. 设置摩擦力为 1.0
2. 使用 Maximum 摩擦组合
3. 添加适当的阻力
4. 考虑冻结旋转
