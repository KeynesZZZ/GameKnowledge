# 标签体系

> 知识库统一标签分类，用于文档分类和检索

## 标签分类

### 按技术领域

| 标签 | 说明 | 示例文档 |
|------|------|----------|
| `#架构` | 系统架构设计 | 单例模式、ECS架构 |
| `#渲染` | 渲染相关技术 | Shader、URP、后处理 |
| `#物理` | 物理系统 | 碰撞检测、刚体 |
| `#动画` | 动画系统 | Animator、Playables |
| `#UI` | 用户界面 | UGUI、UI架构 |
| `#音频` | 音频系统 | AudioSource、混音 |
| `#网络` | 网络通信 | Socket、同步模型 |
| `#AI` | 人工智能 | 行为树、导航 |

### 按优化类型

| 标签 | 说明 | 示例文档 |
|------|------|----------|
| `#性能优化` | 性能相关优化 | DrawCall优化、GC优化 |
| `#内存优化` | 内存相关优化 | 内存泄漏、资源管理 |
| `#加载优化` | 加载相关优化 | 资源加载、场景加载 |

### 按质量属性

| 标签 | 说明 | 示例文档 |
|------|------|----------|
| `#最佳实践` | 推荐做法 | 代码规范、设计模式 |
| `#反模式` | 应避免的做法 | 常见错误、性能陷阱 |
| `#踩坑记录` | 实际遇到的问题 | Bug记录、解决方案 |

### 按文档类型

| 标签 | 说明 | 示例文档 |
|------|------|----------|
| `#教程` | 入门教程 | 新手指南、快速入门 |
| `#深度解析` | 原理分析 | 源码分析、机制讲解 |
| `#代码片段` | 可复用代码 | 工具类、扩展方法 |
| `#工具使用` | 工具使用说明 | Profiler使用、调试技巧 |

### 按平台

| 标签 | 说明 | 示例文档 |
|------|------|----------|
| `#iOS` | iOS平台 | iOS专项、AppStore |
| `#Android` | Android平台 | Android专项、GooglePlay |
| `#WebGL` | WebGL平台 | 浏览器限制、优化 |
| `#主机` | 主机平台 | Switch、PS、Xbox |

### 按状态

| 标签 | 说明 | 使用场景 |
|------|------|----------|
| `#草稿` | 正在编写中 | 未完成的文档 |
| `#待验证` | 需要验证 | 待测试的方案 |
| `#已归档` | 已过时 | 旧版本相关内容 |

## 标签使用规范

1. **每篇文档至少包含2个标签**
2. **优先使用技术领域标签**
3. **添加状态标签便于筛选**
4. **避免创建重复含义的标签**

## 标签组合示例

```markdown
#渲染 #性能优化 #深度解析
```

```markdown
#UI #最佳实践 #教程
```

```markdown
#内存优化 #踩坑记录 #iOS
```
