## 核心知识点

### 1. EditorWindow

```csharp
public class MyWindow : EditorWindow
{
    [MenuItem("Tools/My Window")]
    static void Open() => GetWindow<MyWindow>();

    void OnGUI()
    {
        // 绘制 UI
    }
}
```

### 2. 自定义 Inspector

```csharp
[CustomEditor(typeof(MyComponent))]
public class MyComponentEditor : Editor
{
    public override void OnInspectorGUI()
    {
        // 自定义 Inspector 显示
    }
}
```

### 3. PropertyDrawer

```csharp
[CustomPropertyDrawer(typeof(MyType))]
public class MyTypeDrawer : PropertyDrawer
{
    public override void OnGUI(Rect position, SerializedProperty property, GUIContent label)
    {
        // 自定义属性绘制
    }
}
```

### 4. Gizmos & Handles

| 类型 | 用途 |
|------|------|
| Gizmos.DrawWireCube | 绘制线框 |
| Gizmos.DrawMesh | 绘制网格 |
| Handles.PositionHandle | 位置控制柄 |
| Handles.RadiusHandle | 半径控制柄 |
