# 编辑器扩展学习路径

> 目标：掌握 Unity 编辑器扩展，提升开发效率

## 文档定位

系统性规划Unity编辑器扩展的学习路线，从EditorWindow和自定义Inspector入手，到PropertyDrawer、Gizmos/Handles可视化，最终掌握AssetPostprocessor和构建管线扩展，包含核心概念速查和实用工具示例。

---

## 前置知识

- Unity 基础
- C# 基础
- GUI 编程概念

## 学习路线

```
Week 1: 基础编辑器窗口
    ├── EditorWindow 基类
    ├── GUILayout/EditorGUILayout
    └── 菜单项添加

Week 2: 自定义 Inspector
    ├── Editor 类
    ├── CustomEditor 属性
    └── PropertyDrawer

Week 3: 场景可视化
    ├── Gizmos 类
    ├── Handles 类
    └── 场景 GUI

Week 4: 资源处理
    ├── AssetPostprocessor
    ├── 构建管线扩展
    └── 自动化工具
```

## 核心知识点

### 1. EditorWindow

```csharp
public class MyWindow : EditorWindow
{
    [MenuItem("Tools/My Window")]
    static void Open() => GetWindow<MyWindow>();

    void OnGUI()
    {
        // 绘制 UI
    }
}
```

### 2. 自定义 Inspector

```csharp
[CustomEditor(typeof(MyComponent))]
public class MyComponentEditor : Editor
{
    public override void OnInspectorGUI()
    {
        // 自定义 Inspector 显示
    }
}
```

### 3. PropertyDrawer

```csharp
[CustomPropertyDrawer(typeof(MyType))]
public class MyTypeDrawer : PropertyDrawer
{
    public override void OnGUI(Rect position, SerializedProperty property, GUIContent label)
    {
        // 自定义属性绘制
    }
}
```

### 4. Gizmos & Handles

| 类型 | 用途 |
|------|------|
| Gizmos.DrawWireCube | 绘制线框 |
| Gizmos.DrawMesh | 绘制网格 |
| Handles.PositionHandle | 位置控制柄 |
| Handles.RadiusHandle | 半径控制柄 |
