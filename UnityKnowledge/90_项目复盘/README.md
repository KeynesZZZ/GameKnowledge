# 项目复盘

> 项目经验总结与复盘

## 目录说明

此目录用于存储具体项目的复盘文档，按项目名称分类。

### 建议结构

```
90_项目复盘/
└── [项目名]/
    ├── 技术方案回顾.md
    ├── 踩坑记录.md
    ├── 性能优化总结.md
    └── 经验教训.md
```

## 使用规范

1. 每个项目创建独立子目录
2. 使用模板记录关键信息
3. 及时更新，保持信息准确
4. 提炼可复用的经验到其他目录
