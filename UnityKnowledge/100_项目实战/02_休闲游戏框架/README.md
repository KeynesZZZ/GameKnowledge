# 休闲游戏框架

> 基于 Unity 2022.3 LTS 的休闲游戏开发框架，支持 iOS/Android/小游戏多平台

## 技术栈

| 模块 | 技术选型 | 用途 |
|------|---------|------|
| 资源管理 | YooAssets | 热更新资源、异步加载、内存管理 |
| 配置管理 | Luban | Excel配置表 → C#代码生成 |
| 代码热更 | HybridCLR | C#代码热更新 |
| 异步编程 | UniTask | 零GC异步操作 |
| 动画系统 | DOTween | UI和游戏动画 |

## 目录结构

```
Assets/
├── Game/                      # AOT代码（不可热更）
│   ├── Scripts/
│   │   └── Core/              # 核心框架入口
│   ├── Scenes/
│   └── Prefabs/
│
├── HotUpdate/                 # 热更代码
│   └── Scripts/
│       ├── Core/              # 框架核心（可热更）
│       ├── Game/              # 游戏逻辑
│       └── UI/                # UI逻辑
│
├── AddressableAssets/         # YooAssets资源
├── HybridCLRData/             # HybridCLR热更DLL
└── Resources/                 # 预加载资源
```

## 设计原则

1. **热更优先**：业务逻辑全部放热更DLL，减少发包频率
2. **Prefab纯净**：预制体不挂载脚本，节点变更无需重打包
3. **代码生成**：重复代码自动生成，减少手动编写
4. **配置驱动**：动画、缓存等策略通过配置文件管理
5. **平台适配**：统一接口封装平台差异
