### 3.2 方法论问题详情

#### ⚠️ 问题1: 缺少预热阶段

**影响文档**: foreach vs for, Dictionary性能, 协程vs UniTask

**问题**: 首次运行代码会有JIT编译开销，可能影响结果

**建议**: 在正式测试前运行1-2次预热

```csharp
// 推荐做法
private void Start()
{
    // 预热
    RunBenchmark();
    GC.Collect();

    // 正式测试
    RunBenchmark();
}
```

#### ⚠️ 问题2: 未多次运行取平均

**影响文档**: 全部文档

**问题**: 单次测试可能受系统波动影响

**建议**: 运行3-5次取中位数或平均值

### 5.2 发现的不一致

#### ⚠️ 轻微不一致

**问题**: 【性能数据】纹理压缩格式对比 中 ASTC 4x4 压缩率标注为87.5%

**验证**:
```
ASTC 4x4 = 128 bits per 16 pixels = 8 bits/pixel
RGBA32 = 32 bits/pixel
压缩率 = (32-8)/32 = 75%

文档标注87.5%有误，应为75%
```

**影响**: 轻微，不影响实际使用建议
