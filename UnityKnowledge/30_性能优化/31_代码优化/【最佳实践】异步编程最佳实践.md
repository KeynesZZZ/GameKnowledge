## 二、协程最佳实践

### 2.1 启动和停止

```csharp
// ✅ 正确：保存协程引用
private Coroutine updateCoroutine;

void OnEnable()
{
    updateCoroutine = StartCoroutine(UpdateRoutine());
}

void OnDisable()
{
    if (updateCoroutine != null)
    {
        StopCoroutine(updateCoroutine);
        updateCoroutine = null;
    }
}

IEnumerator UpdateRoutine()
{
    while (true)
    {
        // 更新逻辑
        yield return new WaitForSeconds(1f);
    }
}
```

---

### 2.2 避免内存泄漏

```csharp
// ❌ 错误：忘记停止协程
void Start()
{
    StartCoroutine(LoadDataAsync());
}

IEnumerator LoadDataAsync()
{
    while (true)
    {
        // 加载数据
        yield return new WaitForSeconds(1f);
    }
    // 对象销毁时协程仍在运行！
}

// ✅ 正确：在OnDisable中停止
private Coroutine loadDataCoroutine;

void OnEnable()
{
    loadDataCoroutine = StartCoroutine(LoadDataAsync());
}

void OnDisable()
{
    if (loadDataCoroutine != null)
    {
        StopCoroutine(loadDataCoroutine);
    }
}
```

## 三、UniTask最佳实践

### 3.1 UniTask基础使用

```csharp
using Cysharp.Threading.Tasks;
using UnityEngine;

public class UniTaskExample : MonoBehaviour
{
    private async void Start()
    {
        // 异步加载资源
        GameObject prefab = await Resources.LoadAsync<GameObject>("Prefab");

        // 异步等待
        await Delay(1.0f);

        // UniTask延迟（无GC）
        await UniTask.Delay(1000);

        // 等待帧
        await UniTask.Yield();
        await UniTask.NextFrame();
    }

    private async void Delay(float seconds)
    {
        await UniTask.Delay(TimeSpan.FromSeconds(seconds));
    }
}
```

---

### 3.2 UniTask vs 协程

```csharp
// 协程版本
IEnumerator LoadAssetAsync(string path)
{
    var loadOp = Resources.LoadAsync<GameObject>(path);
    while (!loadOp.isDone)
    {
        yield return null;  // 每帧产生GC
    }
    var prefab = loadOp.asset;
}

// UniTask版本
async UniTask<GameObject> LoadAssetAsync(string path)
{
    var loadOp = Resources.LoadAsync<GameObject>(path);
    await loadOp;  // 无GC
    return loadOp.asset;
}

性能对比：
├─ 协程：每帧24B GC
├─ UniTask：0B GC
└─ 性能提升：显著
```

## 四、异步最佳实践

### 4.1 错误处理

```csharp
// ✅ 正确：处理异步异常
async void LoadSceneAsync(string sceneName)
{
    try
    {
        var loadOp = SceneManager.LoadSceneAsync(sceneName);
        await UniTask.WaitUntil(() => loadOp.isDone);
        Debug.Log("场景加载完成");
    }
    catch (System.Exception e)
    {
        Debug.LogError($"场景加载失败: {e.Message}");
    }
}
```

---

### 4.2 取消异步操作

```csharp
// ✅ 正确：使用CancellationTokenSource
private CancellationTokenSource cts;

void Start()
{
    cts = new CancellationTokenSource();
    LoadDataAsync(cts.Token).Forget();
}

void OnDestroy()
{
    cts?.Cancel();
    cts?.Dispose();
}

async UniTask LoadDataAsync(CancellationToken token)
{
    try
    {
        await UniTask.Delay(5000, token);
    }
    catch (OperationCanceledException)
    {
        Debug.Log("加载已取消");
    }
}
```
