## 二、Shader复杂度性能测试

### 2.1 数学运算性能

```
测试环境：
- GPU：Adreno 650（中端移动GPU）
- 测试对象：1000个全屏四边形
- 测试指标：GPU时间（毫秒）

┌─────────────────────────────────────────────────────────────┐
│              数学运算性能对比                                 │
├─────────────────────────────────────────────────────────────┤
│ 运算类型 │ GPU时间 │ 相对性能 │ 说明                        │
├──────────┼─────────┼─────────┼─────────────────────────────┤
│ 加减法   │ 2.1ms   │ 1.0x    │ 基准，最快                   │
│ 乘除法   │ 2.3ms   │ 1.1x    │ 略慢                         │
│ sin/cos  │ 5.8ms   │ 2.8x    │ 慢2.8倍                     │
│ pow      │ 6.2ms   │ 3.0x    │ 慢3倍                        │
│ sqrt     │ 4.1ms   │ 2.0x    │ 慢2倍                        │
│ log/exp  │ 7.5ms   │ 3.6x    │ 慢3.6倍                     │
└──────────┴─────────┴─────────┴─────────────────────────────┘

结论：
✅ 尽量使用简单运算（加减乘除）
❌ 避免复杂运算（sin、cos、pow、log）
✅ 使用查找表（Lookup Table）替代复杂运算
```

### 4.2 分支优化

```
测试环境：
- GPU：Adreno 650
- 测试场景：1000个像素，50%需要特殊处理

┌─────────────────────────────────────────────────────────────┐
│            分支对性能的影响                                  │
├─────────────────────────────────────────────────────────────┤
│ 分支类型 │ GPU时间 │ 说明                                  │
├──────────┼─────────┼───────────────────────────────────────┤
│ 无分支   │ 3.2ms   │ 所有像素统一处理                    │
│ if分支   │ 5.8ms   │ 导致warp divergence，性能下降82%     │
│ 三元运算 │ 4.5ms   │ 比if好，但仍比无分支慢40%            │
│ step函数 │ 3.5ms   │ 无分支，仅比无分支慢9%              │
└──────────┴─────────┴───────────────────────────────────────┘

代码示例：
// ❌ 使用if
if (distance < 1.0)
{
    color = fixed4(1, 0, 0, 1);
}
else
{
    color = fixed4(0, 0, 1, 1);
}

// ✅ 使用step
fixed4 near = fixed4(1, 0, 0, 1);
fixed4 far = fixed4(0, 0, 1, 1);
color = lerp(far, near, step(1.0 - distance, 0.5));
```

### 5.2 实际优化案例

#### 案例1：简化光照模型

```hlsl
// ❌ 复杂光照（Standard Shader）
Shader "Standard"
{
    // PBR计算
    // 4次纹理采样（Albedo, Normal, Metallic, Smoothness）
    // 复杂光照计算（多次反射）
    // GPU时间：11.5ms
}

// ✅ 简化光照（Mobile Shader）
Shader "Mobile/Diffuse"
{
    // 简单漫反射
    // 1次纹理采样（Albedo）
    // 简化光照计算
    // GPU时间：2.1ms
    // 性能提升：5.5倍
}
```

---

#### 案例2：减少纹理采样

```hlsl
// ❌ 5次纹理采样
sampler2D _MainTex;
sampler2D _BumpMap;
sampler2D _DetailTex;
sampler2D _EmissionMap;
sampler2D _MaskMap;

fixed4 col = tex2D(_MainTex, uv);
fixed3 bump = tex2D(_BumpMap, uv).xyz;
fixed4 detail = tex2D(_DetailTex, uv);
fixed3 emission = tex2D(_EmissionMap, uv).rgb;
fixed mask = tex2D(_MaskMap, uv).r;
// GPU时间：12.8ms

// ✅ 1次纹理采样（使用纹理数组）
sampler2D _MainTex;

fixed4 col = tex2D(_MainTex, uv);
// 其他信息打包在纹理的不同通道
// GPU时间：3.2ms
// 性能提升：4倍
```

---

#### 案例3：使用half精度

```hlsl
// ❌ 使用float
uniform float4 _Color;
uniform float _Strength;
uniform float3 _Position;

// GPU时间：5.2ms

// ✅ 使用half
uniform half4 _Color;
uniform half _Strength;
uniform half3 _Position;

// GPU时间：3.8ms
// 性能提升：27%
```
