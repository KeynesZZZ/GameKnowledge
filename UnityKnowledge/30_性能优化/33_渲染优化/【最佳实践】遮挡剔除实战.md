## 1. 遮挡剔除原理

### 1.1 基本概念

**遮挡剔除（Occlusion Culling）**：预先计算哪些物体会被其他物体遮挡，运行时跳过被遮挡物体的渲染。

```
┌─────────────────────────────────────────────────────────────┐
│                    遮挡剔除原理                              │
│                                                             │
│   视锥剔除 (Frustum Culling)                                │
│   ├── 剔除视野范围外的物体                                  │
│   ├── Unity 自动执行                                        │
│   └── 不需要手动配置                                        │
│                                                             │
│   遮挡剔除 (Occlusion Culling)                              │
│   ├── 剔除视野范围内但被遮挡的物体                          │
│   ├── 需要烘焙数据                                          │
│   └── 适合静态场景                                          │
│                                                             │
│   ┌──────────────┐                                         │
│   │     Camera   │                                         │
│   │      ↓       │                                         │
│   │   ┌─────┐    │  ← 墙壁遮挡后方物体                     │
│   │   │Wall │ ■■ │  ← 这两个方块被遮挡                     │
│   │   └─────┘    │                                         │
│   └──────────────┘                                         │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 适用场景

| 场景类型 | 是否推荐 | 原因 |
|---------|---------|------|
| 室内场景 | ✅ 强烈推荐 | 大量墙壁遮挡 |
| 城市街道 | ✅ 推荐 | 建筑物遮挡 |
| 开阔地形 | ❌ 不推荐 | 几乎没有遮挡 |
| 小型场景 | ❌ 不推荐 | 计算开销大于收益 |
| 动态场景 | ❌ 不适用 | 仅支持静态物体 |

## 2. 遮挡剔除配置

### 2.1 基础配置步骤

```
步骤1：标记静态物体
├── 选中场景中的静态物体
├── Inspector 中勾选 "Occluder Static"（遮挡者）
├── 勾选 "Occludee Static"（被遮挡者）
└── 大型物体（墙壁）应同时勾选两者

步骤2：打开遮挡剔除窗口
├── Window → Rendering → Occlusion Culling
└── 或使用快捷键

步骤3：烘焙遮挡数据
├── 点击 "Bake" 按钮
├── 等待烘焙完成
└── 数据保存在场景中
```

### 2.2 遮挡剔除窗口详解

```
┌─────────────────────────────────────────────────────────────┐
│                 Occlusion Culling 窗口                       │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  [Object] [Bake] [Visualization]                           │
│                                                             │
│  ─────────── Bake Settings ───────────                     │
│                                                             │
│  Smallest Occluder: 5                                      │
│  ├── 最小遮挡物尺寸（单位：米）                             │
│  ├── 小于此值的物体不会作为遮挡者                           │
│  └── 建议：根据场景中最小墙壁尺寸设置                       │
│                                                             │
│  Smallest Hole: 0.25                                       │
│  ├── 最小孔洞尺寸                                           │
│  ├── 小于此值的缝隙会被忽略                                 │
│  └── 建议：门框、窗户尺寸                                   │
│                                                             │
│  Backface Threshold: 100                                   │
│  ├── 背面剔除阈值                                           │
│  ├── 100 = 完全启用                                         │
│  └── 建议：保持默认                                         │
│                                                             │
│  ─────────── Buttons ───────────                           │
│                                                             │
│  [Bake]   - 烘焙遮挡数据                                    │
│  [Clear]  - 清除遮挡数据                                    │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2.3 编程配置

```csharp
#if UNITY_EDITOR
/// <summary>
/// 遮挡剔除自动配置工具
/// </summary>
public static class OcclusionCullingSetup
{
    /// <summary>
    /// 自动配置场景中的静态物体
    /// </summary>
    [MenuItem("Tools/Occlusion/Setup Static Objects")]
    public static void SetupStaticObjects()
    {
        var allRenderers = GameObject.FindObjectsOfType<MeshRenderer>();
        int occluderCount = 0;
        int occludeeCount = 0;

        foreach (var renderer in allRenderers)
        {
            var go = renderer.gameObject;

            // 判断是否为大型物体（墙壁、地板等）
            if (IsLargeObject(renderer))
            {
                // 大型物体：同时是遮挡者和被遮挡者
                GameObjectUtility.SetStaticEditorFlags(go,
                    StaticEditorFlags.OccluderStatic | StaticEditorFlags.OccludeeStatic);
                occluderCount++;
            }
            else if (renderer.gameObject.isStatic)
            {
                // 小型静态物体：仅是被遮挡者
                GameObjectUtility.SetStaticEditorFlags(go,
                    StaticEditorFlags.OccludeeStatic);
                occludeeCount++;
            }
        }

        Debug.Log($"配置完成: {occluderCount} 个遮挡者, {occludeeCount} 个被遮挡者");
    }

    private static bool IsLargeObject(MeshRenderer renderer)
    {
        var bounds = renderer.bounds;
        float volume = bounds.size.x * bounds.size.y * bounds.size.z;

        // 大于 10 立方米视为大型物体
        return volume > 10f;
    }

    /// <summary>
    /// 烘焙遮挡数据
    /// </summary>
    [MenuItem("Tools/Occlusion/Bake Occlusion Data")]
    public static void BakeOcclusionData()
    {
        // 注意：这里使用反射调用内部 API
        // 实际使用时建议手动在 Occlusion 窗口烘焙

        var staticFlags = StaticEditorFlags.OccluderStatic | StaticEditorFlags.OccludeeStatic;
        UnityEditor.SceneManagement.EditorSceneManager.MarkSceneDirty(
            UnityEditor.SceneManagement.EditorSceneManager.GetActiveScene());

        Debug.Log("请在 Window → Rendering → Occlusion Culling 中点击 Bake");
    }
}
#endif
```

## 3. 烘焙优化

### 3.1 烘焙参数调优

```csharp
/// <summary>
/// 遮挡剔除烘焙参数配置
/// </summary>
public class OcclusionBakeSettings : MonoBehaviour
{
    [Header("Scene Analysis")]
    [SerializeField] private float sceneSize = 100f;
    [SerializeField] private float minWallThickness = 0.3f;
    [SerializeField] private float smallestDoorWidth = 1f;

    /// <summary>
    /// 计算推荐的烘焙参数
    /// </summary>
    [ContextMenu("Calculate Recommended Settings")]
    public void CalculateRecommendedSettings()
    {
        // Smallest Occluder 建议
        // 应该略大于场景中最薄的墙壁厚度
        float recommendedOccluder = minWallThickness * 1.5f;

        // Smallest Hole 建议
        // 应该等于或小于最小的门/窗尺寸
        float recommendedHole = smallestDoorWidth * 0.5f;

        Debug.Log($@"
推荐烘焙参数：
├── Smallest Occluder: {recommendedOccluder:F2}
├── Smallest Hole: {recommendedHole:F2}
└── Backface Threshold: 100
        ");
    }
}
```

### 3.3 分区烘焙

```csharp
/// <summary>
/// 大型场景分区烘焙
/// </summary>
public class ZoneBasedOcclusion : MonoBehaviour
{
    [Header("Zone Configuration")]
    [SerializeField] private Transform[] zoneCenters;
    [SerializeField] private float zoneRadius = 50f;

    /// <summary>
    /// 根据玩家位置启用/禁用遮挡区域
    /// </summary>
    public void UpdateActiveZones(Vector3 playerPosition)
    {
        foreach (var zone in zoneCenters)
        {
            float distance = Vector3.Distance(playerPosition, zone.position);
            bool isActive = distance < zoneRadius * 2f;

            // 启用/禁用该区域的遮挡数据
            var occlusionPortal = zone.GetComponent<OcclusionPortal>();
            if (occlusionPortal != null)
            {
                occlusionPortal.open = !isActive;
            }
        }
    }
}
```

## 4. Occlusion Portal 使用

### 4.1 Portal 概念

**Occlusion Portal**：用于动态控制遮挡剔除的"门"，适合需要动态开/关遮挡的场景。

```
┌─────────────────────────────────────────────────────────────┐
│                    Occlusion Portal                          │
│                                                             │
│   场景：                                                    │
│   ┌─────────────────┐                                      │
│   │   Room A        │   Door    │   Room B        │        │
│   │                 │ ┌─────┐ │ │                 │        │
│   │    Player       │ │     │ │ │  (不可见)       │        │
│   │       ●         │ │     │ │ │                 │        │
│   │                 │ └─────┘ │ │                 │        │
│   └─────────────────┘         └─────────────────┘        │
│                                                             │
│   Door 关闭 (Portal.open = false):                         │
│   └── Room B 被遮挡剔除                                     │
│                                                             │
│   Door 打开 (Portal.open = true):                          │
│   └── Room B 可见，正常渲染                                 │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 4.2 Portal 配置

```csharp
/// <summary>
/// 遮挡门控制器
/// </summary>
[RequireComponent(typeof(OcclusionPortal))]
public class OcclusionDoorController : MonoBehaviour
{
    [Header("Door Settings")]
    [SerializeField] private Animator doorAnimator;
    [SerializeField] private string openParameter = "IsOpen";
    [SerializeField] private float transitionDuration = 0.5f;

    private OcclusionPortal portal;
    private bool isOpen = false;

    private void Awake()
    {
        portal = GetComponent<OcclusionPortal>();
    }

    /// <summary>
    /// 打开门
    /// </summary>
    public void OpenDoor()
    {
        if (isOpen) return;

        isOpen = true;
        doorAnimator.SetBool(openParameter, true);

        // 延迟开启 Portal（等待动画播放）
        StartCoroutine(DelayedPortalOpen(transitionDuration));
    }

    /// <summary>
    /// 关闭门
    /// </summary>
    public void CloseDoor()
    {
        if (!isOpen) return;

        isOpen = false;
        doorAnimator.SetBool(openParameter, false);

        // 立即关闭 Portal
        portal.open = false;
    }

    private IEnumerator DelayedPortalOpen(float delay)
    {
        yield return new WaitForSeconds(delay);
        portal.open = true;
    }

    /// <summary>
    /// 根据玩家距离自动开关门
    /// </summary>
    public void AutoControlByDistance(Transform player, float openDistance)
    {
        float distance = Vector3.Distance(transform.position, player.position);

        if (distance < openDistance && !isOpen)
        {
            OpenDoor();
        }
        else if (distance > openDistance * 1.5f && isOpen)
        {
            CloseDoor();
        }
    }
}
```

### 5.2 运行时调试

```csharp
/// <summary>
/// 遮挡剔除调试工具
/// </summary>
public class OcclusionDebugTool : MonoBehaviour
{
    [Header("Debug Settings")]
    [SerializeField] private bool showDebugInfo = true;
    [SerializeField] private Camera targetCamera;

    private int totalRenderers;
    private int visibleRenderers;
    private int culledRenderers;

    private void Start()
    {
        if (targetCamera == null)
            targetCamera = Camera.main;
    }

    private void Update()
    {
        if (!showDebugInfo) return;

        UpdateStats();
    }

    private void UpdateStats()
    {
        var allRenderers = GameObject.FindObjectsOfType<Renderer>();
        totalRenderers = allRenderers.Length;

        visibleRenderers = 0;
        foreach (var renderer in allRenderers)
        {
            if (renderer.isVisible)
                visibleRenderers++;
        }

        culledRenderers = totalRenderers - visibleRenderers;
    }

    private void OnGUI()
    {
        if (!showDebugInfo) return;

        GUILayout.BeginArea(new Rect(10, 150, 250, 120));
        GUILayout.Label("=== Occlusion Culling Stats ===");
        GUILayout.Label($"Total Renderers: {totalRenderers}");
        GUILayout.Label($"Visible: {visibleRenderers}");
        GUILayout.Label($"Culled: {culledRenderers}");

        float cullPercent = totalRenderers > 0 ? (float)culledRenderers / totalRenderers * 100f : 0f;
        GUILayout.Label($"Cull Rate: {cullPercent:F1}%");
        GUILayout.EndArea();
    }

    /// <summary>
    /// 可视化被剔除的物体
    /// </summary>
    [ContextMenu("Visualize Culled Objects")]
    public void VisualizeCulledObjects()
    {
        var allRenderers = GameObject.FindObjectsOfType<Renderer>();

        foreach (var renderer in allRenderers)
        {
            if (!renderer.isVisible && renderer.gameObject.isStatic)
            {
                Debug.DrawLine(
                    renderer.bounds.center,
                    targetCamera.transform.position,
                    Color.red,
                    5f
                );
            }
        }
    }
}
```
