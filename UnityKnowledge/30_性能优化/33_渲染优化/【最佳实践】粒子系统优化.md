# 【最佳实践】粒子系统优化

## 文档定位

Unity粒子系统性能优化指南，涵盖粒子数量控制（移动端限制建议）、碰撞优化（使用平面碰撞替代网格碰撞）、渲染优化（使用简单Shader、减少纹理采样）、粒子LOD系统、对象池复用粒子特效，以及GPU粒子与CPU粒子的选择策略。

---

## 核心建议

```
粒子系统优化原则：
1. 限制粒子数量
2. 减少粒子寿命
3. 避免粒子碰撞
4. 使用GPU粒子
5. 合理使用预加热
```

---

## 一、粒子数量优化

### 1.1 数量限制

```
性能数据：
├─ 100个粒子：可忽略
├─ 1000个粒子：轻微影响
├─ 5000个粒子：明显影响
└─ 10000个粒子：严重性能问题

建议：
├─ 移动平台：<1000个粒子/系统
├─ PC平台：<5000个粒子/系统
└─ 总粒子数：<10000个
```

---

## 二、粒子碰撞优化

```
┌─────────────────────────────────────────────────────────────┐
│            粒子碰撞性能对比                                  │
├─────────────────────────────────────────────────────────────┤
│ 碰撞类型      │ 性能  │ 适用场景                          │
├──────────────┼───────┼───────────────────────────────────┤
│ 无碰撞        │ 最快 │ 视觉效果（默认）                  │
│ World碰撞    │ 慢    │ 地面、墙壁（少用）                │
│ Plane碰撞    │ 中    │ 简单平面                          │
│ 粒子间碰撞   │ 最慢 │ 极少使用                          │
└──────────────┴───────┴───────────────────────────────────┘

建议：
✅ 尽量使用无碰撞
❌ 避免粒子间碰撞
✅ 移动平台禁用碰撞
```

---

## 三、渲染优化

```
优化技巧：
1. 减少粒子大小
2. 使用最短寿命
3. 禁用预加热（非必要）
4. 使用GPU粒子（移动平台）
5. 减少发射率

性能提升：
├─ 优化前：50ms（5000粒子）
├─ 优化后：5ms（500粒子）
└─ 提升：10倍
```

```

□ 使用GPU粒子
□ 减少发射率

```
