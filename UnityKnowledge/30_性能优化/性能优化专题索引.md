## 按场景查找性能问题

### 场景1：游戏卡顿、帧率低

**可能原因**：
1. CPU瓶颈 → [CPU优化技术](./【教程】CPU优化技术.md)
2. 渲染瓶颈 → [渲染优化](./33_渲染优化/)
3. GC卡顿 → [GC优化清单](./32_内存管理/【最佳实践】GC优化清单.md)

**诊断步骤**：
```
1. 打开Profiler
2. 检查CPU占用 → 如果高，查看代码优化
3. 检查渲染占用 → 如果高，查看渲染优化
4. 检查GC.Alloc → 如果有，查看内存管理
```

## 性能优化流程

### 标准流程

```
1. 基准测试
   ├─ 记录当前性能指标
   ├─ 帧率、内存、DrawCall
   └─ 建立性能基线

2. 问题定位
   ├─ 使用Profiler分析
   ├─ 找到性能瓶颈
   └─ 确定优化方向

3. 实施优化
   ├─ 应用优化技巧
   ├─ 测试优化效果
   └─ 验证功能正常

4. 验证结果
   ├─ 对比优化前后数据
   ├─ 确认性能提升
   └─ 记录优化经验
```
