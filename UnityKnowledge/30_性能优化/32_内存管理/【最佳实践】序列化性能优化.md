## 二、JsonUtility优化

### 2.1 基础使用

```csharp
// ✅ 正确：使用JsonUtility
string json = JsonUtility.ToJson(data);
Data data = JsonUtility.FromJson<Data>(json);

// ❌ 错误：频繁调用
void Update()
{
    string json = JsonUtility.ToJson(playerData);  // 每帧序列化
    // 性能灾难
}
```

---

### 2.2 优化技巧

```
✅ 使用技巧：
1. 缓存序列化结果
2. 只在数据变化时序列化
3. 使用StringBuilder拼接
4. 避免循环引用

❌ 避免错误：
1. Update中序列化
2. 序列化大对象（>1MB）
3. 频繁序列化
```

## 三、二进制序列化

### 3.1 BinaryFormatter使用

```csharp
using System;
using System.IO;
using System.Runtime.Serialization.Formatters.Binary;

public class BinarySerializer
{
    public static byte[] Serialize<T>(T obj)
    {
        using (var ms = new MemoryStream())
        {
            var formatter = new BinaryFormatter();
            formatter.Serialize(ms, obj);
            return ms.ToArray();
        }
    }

    public static T Deserialize<T>(byte[] data)
    {
        using (var ms = new MemoryStream(data))
        {
            var formatter = new BinaryFormatter();
            return (T)formatter.Deserialize(ms);
        }
    }
}
```

---

### 3.2 性能优化

```
优化建议：
✅ 使用二进制序列化（性能提升3倍）
✅ 使用Protobuf（性能提升5倍）
✅ 减少序列化频率
✅ 只序列化必要数据

性能提升：
├─ JsonUtility → BinaryFormatter：2.7倍
├─ JsonUtility → Protobuf：4倍
└─ 内存分配显著减少
```
