## 设计原则

### 编号规则

| 前缀 | 分类 | 说明 |
|------|------|------|
| 00 | 元数据 | 模板、配置、规范 |
| 10-30 | 技术知识 | 按技术领域分类 |
| 40-60 | 工具与库 | 开发工具、平台、第三方库 |
| 90 | 经验总结 | 项目复盘、踩坑记录 |

### 文档规范

#### YAML Frontmatter（推荐）

所有新文档应使用 YAML frontmatter：

```yaml
---
title: 【代码片段】对象池通用实现
tags: [C#, Unity, 架构, 性能优化, 代码片段]
category: 架构设计/代码片段
created: 2024-01-15 10:30
updated: 2024-03-04 15:20
description: C#泛型对象池的基础实现，减少GC分配
unity_version: 2021.3+
---
```

## 快速开始

### 查找知识

1. 按目录导航
2. 使用知识库搜索（ChromaDB）
3. 按标签筛选
4. 使用 Obsidian 双向链接跳转

### 贡献知识

1. 使用对应模板创建文档
2. 添加 YAML frontmatter
3. 添加适当标签
4. 使用 `[[笔记名]]` 格式添加内部链接
5. 更新相关 README
