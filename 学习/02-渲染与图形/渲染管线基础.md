# 渲染管线基础

> 第1课 | 渲染与图形模块

## 1. 什么是渲染管线？

**渲染管线**是GPU将3D场景转换为2D图像的一系列步骤。

```
┌─────────────────────────────────────────────────────────────┐
│                     渲染管线流程                             │
│                                                             │
│   ┌─────────┐    ┌─────────┐    ┌─────────┐              │
│   │ 3D场景  │ →  │ 几何处理 │ →  │ 光栅化  │              │
│   │(Mesh+Mat)│    │(顶点变换)│    │(三角形→像素)│           │
│   └─────────┘    └─────────┘    └────┬────┘              │
│                                       │                    │
│                                       ↓                    │
│   ┌─────────┐    ┌─────────┐    ┌─────────┐              │
│   │ 最终图像 │ ←  │ 后处理  │ ←  │ 像素处理 │              │
│   │ (屏幕)  │    │(特效)   │    │(着色)   │              │
│   └─────────┘    └─────────┘    └─────────┘              │
└─────────────────────────────────────────────────────────────┘
```

**管线的主要阶段**：

| 阶段 | 工作内容 |
|------|----------|
| **Culling（剔除）** | 决定哪些物体需要渲染 |
| **Rendering（渲染）** | 绘制物体到渲染目标 |
| **Post-processing（后处理）** | 对渲染结果应用效果 |

## 2. Built-in vs URP vs HDRP 对比

```
┌─────────────────────────────────────────────────────────────┐
│                   Unity渲染管线对比                          │
│                                                             │
│   Built-in RP          URP                   HDRP           │
│   ┌─────────┐         ┌─────────┐          ┌─────────┐    │
│   │  传统    │         │  通用    │          │  高清    │    │
│   │  管线    │         │  管线    │          │  管线    │    │
│   └─────────┘         └─────────┘          └─────────┘    │
│                                                             │
│   ✅ 兼容性好          ✅ 性能优秀           ✅ 画质最高    │
│   ✅ 资源丰富          ✅ 跨平台             ✅ 物理渲染    │
│   ❌ 不再更新          ✅ Shader Graph       ✅ 光线追踪    │
│   ❌ 扩展困难          ✅ 移动端友好         ❌ 配置复杂    │
│                       ❌ 功能少于HDRP       ❌ 性能要求高  │
│                                                             │
│   适用场景：            适用场景：            适用场景：      │
│   - 老项目维护          - 移动游戏           - PC/主机     │
│   - 快速原型            - 跨平台游戏         - 3A级画质    │
│   - 学习基础            - 独立游戏           - 写实风格    │
└─────────────────────────────────────────────────────────────┘
```

### 详细对比

| 特性 | Built-in | URP | HDRP |
|------|----------|-----|------|
| 平台支持 | 全平台 | 全平台 | PC/主机 |
| 渲染路径 | Forward/Deferred | Forward+/Deferred | Deferred Only |
| SRP Batcher | ❌ | ✅ | ✅ |
| Shader Graph | ❌ | ✅ | ✅ |
| VFX Graph | ❌ | ✅ | ✅ |
| 光照质量 | 中 | 高 | 最高 |
| 阴影质量 | 中 | 高 | 最高 |
| 移动端性能 | 中 | 优秀 | 不推荐 |

### 选择建议

- **三消游戏（移动端）**：**URP** ✅
- PC端写实游戏：HDRP
- 老项目维护：Built-in

### URP资产配置

```
URP Asset 主要设置：
├── General
│   ├── Depth Texture（深度图）
│   ├── Opaque Texture（不透明图）
│   └── HDR（高动态范围）
│
├── Quality
│   ├── HDR Color Buffer
│   ├── MSAA（多重采样抗锯齿）
│   └── Render Scale（渲染缩放）
│
├── Lighting
│   ├── Main Light（主光源）
│   ├── Additional Lights（额外光源）
│   └── Shadows（阴影）
│
├── Shadows
│   ├── Max Distance
│   ├── Working Unit
│   └── Cascade Count
│
└── Post-processing
    ├── Grading Mode
    └── LUT Size
```

### 默认Forward Renderer Pass顺序

1. **Depth Prepass** - 预处理深度
2. **Draw Opaque** - 绘制不透明物体
3. **Draw Skybox** - 绘制天空盒
4. **Draw Transparent** - 绘制透明物体
5. **Post Process** - 后处理

## 4. Renderer Feature与Render Pass

**Renderer Feature**：URP的可扩展机制，用于添加自定义渲染效果

### 自定义Renderer Feature示例

```csharp
using UnityEngine;
using UnityEngine.Rendering;
using UnityEngine.Rendering.Universal;

/// <summary>
/// 自定义Renderer Feature
/// </summary>
public class CustomRenderFeature : ScriptableRendererFeature
{
    [System.Serializable]
    public class Settings
    {
        public RenderPassEvent renderPassEvent = RenderPassEvent.AfterRenderingOpaques;
        public Material blitMaterial;
        public string textureName = "_CustomTexture";
    }

    [SerializeField] private Settings settings = new Settings();
    private CustomRenderPass renderPass;

    public override void Create()
    {
        renderPass = new CustomRenderPass(settings);
    }

    public override void AddRenderPasses(ScriptableRenderer renderer, ref RenderingData renderingData)
    {
        renderer.EnqueuePass(renderPass);
    }
}

/// <summary>
/// 自定义渲染通道
/// </summary>
public class CustomRenderPass : ScriptableRenderPass
{
    private CustomRenderFeature.Settings settings;
    private RenderTargetIdentifier source;
    private RenderTargetHandle tempTexture;

    public CustomRenderPass(CustomRenderFeature.Settings settings)
    {
        this.settings = settings;
        tempTexture.Init("_TempTexture");
    }

    public override void OnCameraSetup(CommandBuffer cmd, ref RenderingData renderingData)
    {
        source = renderingData.cameraData.renderer.cameraColorTarget;
    }

    public override void Execute(ScriptableRenderContext context, ref RenderingData renderingData)
    {
        if (settings.blitMaterial == null) return;

        CommandBuffer cmd = CommandBufferPool.Get("Custom Pass");

        RenderTextureDescriptor descriptor = renderingData.cameraData.cameraTargetDescriptor;
        descriptor.depthBufferBits = 0;

        cmd.GetTemporaryRT(tempTexture.id, descriptor);
        cmd.Blit(source, tempTexture.Identifier(), settings.blitMaterial);
        cmd.Blit(tempTexture.Identifier(), source);

        context.ExecuteCommandBuffer(cmd);
        CommandBufferPool.Release(cmd);
    }

    public override void OnCameraCleanup(CommandBuffer cmd)
    {
        cmd.ReleaseTemporaryRT(tempTexture.id);
    }
}
```

### 添加Renderer Feature

1. 找到URP Renderer Data资产（通常在 `Settings/ForwardRenderer.asset`）
2. 点击 "Add Renderer Feature"
3. 选择内置或自定义Feature

### 内置Renderer Features

- Screen Space Ambient Occlusion (SSAO)
- Screen Space Shadows
- Bloom
- 自定义Feature...

## 5. 三消游戏渲染架构建议

```
┌─────────────────────────────────────────────────────────────┐
│                  三消游戏渲染架构                            │
│                                                             │
│   URP配置：                                                 │
│   ├── Render Scale: 1.0 (或0.8用于低端设备)                │
│   ├── MSAA: 4x (或2x)                                      │
│   ├── Depth Texture: 开启                                  │
│   ├── Opaque Texture: 关闭 (2D游戏不需要)                  │
│   └── HDR: 关闭 (节省性能)                                 │
│                                                             │
│   光照设置：                                                │
│   ├── Main Light: Sun (简单的2D光照)                       │
│   ├── Additional Lights: Per Pixel (少量)                  │
│   └── Shadows: None (2D游戏通常不需要)                     │
│                                                             │
│   Renderer Features：                                       │
│   ├── 自定义后处理（颜色调整）                              │
│   └── UI特效（可选）                                        │
│                                                             │
│   Shader需求：                                              │
│   ├── 棋盘格子Shader（高亮、选中）                         │
│   ├── 消除特效Shader（溶解、闪烁）                         │
│   └── UI Shader（流光、遮罩）                              │
└─────────────────────────────────────────────────────────────┘
```
