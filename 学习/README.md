## 技能等级对照

| 技能 | 初级(1-2年) | 中级(3-5年) | 高级(5-8年) | 专家(10年+) |
|------|------------|------------|------------|-------------|
| C# | 基础语法 | 泛型、委托 | 反射、异步 | 底层原理、高性能 |
| Unity | 组件使用 | 系统掌握 | 原理理解 | 源码级理解 |
| 渲染 | Shader基础 | 自定义Shader | 管线定制 | 引擎级开发 |
| 架构 | 模式应用 | 模块设计 | 系统架构 | 技术决策 |
| 性能 | 问题定位 | 优化实施 | 系统调优 | 极致优化 |
