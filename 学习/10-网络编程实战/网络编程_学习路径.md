## 核心知识点

### 1. 协议选择

| 协议 | 特点 | 适用场景 |
|------|------|----------|
| TCP | 可靠、有序 | 回合制、聊天 |
| UDP | 快速、无连接 | 实时对战 |
| KCP | 可靠 UDP | 动作游戏 |
| WebSocket | 双向、穿透 | H5 游戏 |

### 2. 同步模型

| 模型 | 带宽 | 一致性 | 实现难度 |
|------|------|--------|----------|
| 状态同步 | 高 | 高 | 中 |
| 帧同步 | 低 | 高 | 高 |
| 混合同步 | 中 | 中 | 高 |

### 3. 关键技术

| 技术 | 说明 |
|------|------|
| 客户端预测 | 减少延迟感 |
| 服务器回滚 | 校正状态 |
| 插值 | 平滑显示 |
| 延迟补偿 | 公平性 |

### 4. Protobuf 示例

```protobuf
message PlayerState {
    int32 player_id = 1;
    float x = 2;
    float y = 3;
    float z = 4;
    int32 hp = 5;
}
```
