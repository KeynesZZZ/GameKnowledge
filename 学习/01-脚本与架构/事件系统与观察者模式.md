# 事件系统与观察者模式

> 第4课 | 脚本与架构模块

## 1. 观察者模式核心概念

**定义**：定义对象间的一对多依赖关系，当一个对象状态改变时，所有依赖它的对象都会收到通知。

```
┌─────────────┐          ┌─────────────┐
│  Subject    │          │  Observer   │
│  (被观察者)  │──────────│  (观察者)    │
│             │  通知     │             │
└─────────────┘          └─────────────┘
       │                        │
       │  ┌─────────────────────┼─────────────────────┐
       │  │                     │                     │
       ▼  ▼                     ▼                     ▼
┌─────────────┐          ┌─────────────┐      ┌─────────────┐
│ Observer 1  │          │ Observer 2  │      │ Observer 3  │
└─────────────┘          └─────────────┘      └─────────────┘
```

**在Unity中的应用**：
- UI更新（分数变化 → UI显示）
- 游戏事件（敌人死亡 → 奖励、统计）
- 输入系统（点击 → 多个响应）

## 2. C# delegate与event

### delegate基础

```csharp
// delegate是类型安全的函数指针
public delegate void ScoreChangedHandler(int newScore);

public class ScoreManager
{
    // delegate变量
    public ScoreChangedHandler OnScoreChanged;

    public void AddScore(int amount)
    {
        // ... 计算分数
        OnScoreChanged?.Invoke(currentScore);
    }
}

// 使用
scoreManager.OnScoreChanged += HandleScoreChanged;
scoreManager.OnScoreChanged -= HandleScoreChanged; // 取消订阅
```

### event关键字的作用

```csharp
public class ScoreManager
{
    // ❌ 没有event：外部可以直接调用和覆盖
    public ScoreChangedHandler OnScoreChanged;

    // ✅ 有event：外部只能 += -=，不能直接调用
    public event ScoreChangedHandler OnScoreChanged;
}

// 区别：
// 1. event只能在声明类内部触发
// 2. event不允许外部直接赋值 (=)
// 3. event提供更好的封装性
```

### 完整的事件实现

```csharp
/// <summary>
/// 事件参数基类
/// </summary>
public class GameEventArgs : System.EventArgs
{
    public float Timestamp { get; }

    public GameEventArgs()
    {
        Timestamp = Time.time;
    }
}

/// <summary>
/// 分数变化事件参数
/// </summary>
public class ScoreChangedEventArgs : GameEventArgs
{
    public int OldScore { get; }
    public int NewScore { get; }
    public int Delta { get; }

    public ScoreChangedEventArgs(int oldScore, int newScore)
    {
        OldScore = oldScore;
        NewScore = newScore;
        Delta = newScore - oldScore;
    }
}

/// <summary>
/// 标准事件模式
/// </summary>
public class ScoreManager
{
    private int currentScore;

    // 使用EventHandler<T>泛型委托
    public event EventHandler<ScoreChangedEventArgs> ScoreChanged;

    public int Score
    {
        get => currentScore;
        set
        {
            if (currentScore != value)
            {
                var oldScore = currentScore;
                currentScore = value;
                OnScoreChanged(new ScoreChangedEventArgs(oldScore, currentScore));
            }
        }
    }

    protected virtual void OnScoreChanged(ScoreChangedEventArgs e)
    {
        ScoreChanged?.Invoke(this, e);
    }
}

// 使用示例
public class UIController : MonoBehaviour
{
    [SerializeField] private Text scoreText;

    private void OnEnable()
    {
        ScoreManager.Instance.ScoreChanged += OnScoreChanged;
    }

    private void OnDisable()
    {
        ScoreManager.Instance.ScoreChanged -= OnScoreChanged;
    }

    private void OnScoreChanged(object sender, ScoreChangedEventArgs e)
    {
        scoreText.text = $"Score: {e.NewScore}";
        Debug.Log($"Score changed: {e.OldScore} → {e.NewScore} (Δ{e.Delta})");
    }
}
```

## 3. UnityEvent vs C# event

### UnityEvent特点

```csharp
using UnityEngine;
using UnityEngine.Events;

public class UnityEventExample : MonoBehaviour
{
    // UnityEvent可以在Inspector中配置
    [Serializable]
    public class IntEvent : UnityEvent<int> { }

    [SerializeField] private IntEvent OnScoreChanged;

    public void AddScore(int amount)
    {
        // ...
        OnScoreChanged?.Invoke(currentScore);
    }
}
```

### 对比分析

| 特性 | C# event | UnityEvent |
|------|----------|------------|
| Inspector配置 | ❌ 不支持 | ✅ 支持 |
| 性能 | ⚡ 快 | 🐢 慢（反射） |
| 序列化 | ❌ 不序列化 | ✅ 可序列化 |
| 多参数 | ✅ 任意数量 | ⚠️ 最多4个参数 |
| 泛型支持 | ✅ 完全支持 | ⚠️ 需要继承 |
| 调试友好 | ❌ 运行时才能看 | ✅ Inspector可见 |

### 选择建议

```csharp
// ✅ 使用C# event的场景
// - 纯代码逻辑
// - 高频调用（每帧）
// - 需要泛型支持
// - 性能敏感

// ✅ 使用UnityEvent的场景
// - 需要策划/设计师配置
// - 简单的UI回调
// - 原型开发
// - 需要Inspector调试

public class HybridExample : MonoBehaviour
{
    // 高频事件用C# event
    public event Action<Vector2> OnMoveInput;

    // 配置性事件用UnityEvent
    [SerializeField] private UnityEvent OnPlayerDeath;
    [SerializeField] private UnityEvent<int> OnLevelComplete;
}
```

## 4. ScriptableObject事件总线

**问题**：跨场景、跨预制体通信困难

**解决方案**：使用ScriptableObject作为事件通道

```csharp
/// <summary>
/// ScriptableObject事件基类
/// </summary>
public abstract class SOEventBase : ScriptableObject
{
    public abstract void Raise();
}

/// <summary>
/// 无参数事件
/// </summary>
[CreateAssetMenu(fileName = "GameEvent", menuName = "Events/Game Event")]
public class GameEvent : SOEventBase
{
    private readonly List<GameEventListener> listeners = new();

    public override void Raise()
    {
        for (int i = listeners.Count - 1; i >= 0; i--)
        {
            listeners[i].OnEventRaised();
        }
    }

    public void RegisterListener(GameEventListener listener)
    {
        if (!listeners.Contains(listener))
            listeners.Add(listener);
    }

    public void UnregisterListener(GameEventListener listener)
    {
        listeners.Remove(listener);
    }
}

/// <summary>
/// 泛型事件（支持参数）
/// </summary>
[CreateAssetMenu(fileName = "GameEvent", menuName = "Events/Game Event (Generic)")]
public class GameEvent<T> : ScriptableObject
{
    private readonly List<IGameEventListener<T>> listeners = new();

    public void Raise(T value)
    {
        for (int i = listeners.Count - 1; i >= 0; i--)
        {
            listeners[i].OnEventRaised(value);
        }
    }

    public void RegisterListener(IGameEventListener<T> listener)
    {
        if (!listeners.Contains(listener))
            listeners.Add(listener);
    }

    public void UnregisterListener(IGameEventListener<T> listener)
    {
        listeners.Remove(listener);
    }
}

/// <summary>
/// 事件监听器接口
/// </summary>
public interface IGameEventListener<T>
{
    void OnEventRaised(T value);
}

/// <summary>
/// MonoBehaviour事件监听器
/// </summary>
public class GameEventListener : MonoBehaviour, IGameEventListener<object>
{
    [SerializeField] private GameEvent<object> gameEvent;
    [SerializeField] private UnityEvent<object> response;

    private void OnEnable()
    {
        gameEvent?.RegisterListener(this);
    }

    private void OnDisable()
    {
        gameEvent?.UnregisterListener(this);
    }

    public void OnEventRaised(object value)
    {
        response?.Invoke(value);
    }
}
```

### 事件总线模式

```csharp
/// <summary>
/// 全局事件总线
/// </summary>
public static class EventBus
{
    private static readonly Dictionary<Type, Delegate> events = new();

    /// <summary>
    /// 订阅事件
    /// </summary>
    public static void Subscribe<T>(Action<T> callback) where T : struct
    {
        var type = typeof(T);
        if (events.TryGetValue(type, out var existing))
        {
            events[type] = Delegate.Combine(existing, callback);
        }
        else
        {
            events[type] = callback;
        }
    }

    /// <summary>
    /// 取消订阅
    /// </summary>
    public static void Unsubscribe<T>(Action<T> callback) where T : struct
    {
        var type = typeof(T);
        if (events.TryGetValue(type, out var existing))
        {
            var newDelegate = Delegate.Remove(existing, callback);
            if (newDelegate == null)
            {
                events.Remove(type);
            }
            else
            {
                events[type] = newDelegate;
            }
        }
    }

    /// <summary>
    /// 发布事件
    /// </summary>
    public static void Publish<T>(T eventData) where T : struct
    {
        var type = typeof(T);
        if (events.TryGetValue(type, out var del))
        {
            (del as Action<T>)?.Invoke(eventData);
        }
    }

    /// <summary>
    /// 清空所有事件
    /// </summary>
    public static void Clear()
    {
        events.Clear();
    }
}

/// <summary>
/// 事件数据结构
/// </summary>
public struct MatchFoundEvent
{
    public int X { get; set; }
    public int Y { get; set; }
    public int Count { get; set; }
    public int GemId { get; set; }
}

public struct ScoreChangedEvent
{
    public int OldScore { get; set; }
    public int NewScore { get; set; }
}

public struct GameStateChangedEvent
{
    public GameState OldState { get; set; }
    public GameState NewState { get; set; }
}

// 使用示例
public class MatchDetector : MonoBehaviour
{
    private void OnMatchFound(int x, int y, int count, int gemId)
    {
        // 发布事件
        EventBus.Publish(new MatchFoundEvent
        {
            X = x,
            Y = y,
            Count = count,
            GemId = gemId
        });
    }
}

public class ScoreManager : MonoBehaviour
{
    private void OnEnable()
    {
        EventBus.Subscribe<MatchFoundEvent>(OnMatchFound);
    }

    private void OnDisable()
    {
        EventBus.Unsubscribe<MatchFoundEvent>(OnMatchFound);
    }

    private void OnMatchFound(MatchFoundEvent e)
    {
        int points = e.Count * 100;
        currentScore += points;

        EventBus.Publish(new ScoreChangedEvent
        {
            OldScore = currentScore - points,
            NewScore = currentScore
        });
    }
}

public class UIController : MonoBehaviour
{
    private void OnEnable()
    {
        EventBus.Subscribe<ScoreChangedEvent>(OnScoreChanged);
    }

    private void OnDisable()
    {
        EventBus.Unsubscribe<ScoreChangedEvent>(OnScoreChanged);
    }

    private void OnScoreChanged(ScoreChangedEvent e)
    {
        scoreText.text = $"Score: {e.NewScore}";
    }
}
```

## 5. 三消游戏实战：事件系统设计

```csharp
/// <summary>
/// 三消游戏所有事件定义
/// </summary>
public static class Match3Events
{
    /// <summary>
    /// 消除事件
    /// </summary>
    public struct MatchFound
    {
        public List<Vector2Int> Positions;
        public int GemId;
        public int MatchCount;
    }

    /// <summary>
    /// 棋子移动事件
    /// </summary>
    public struct GemMoved
    {
        public Vector2Int From;
        public Vector2Int To;
        public int GemId;
    }

    /// <summary>
    /// 战斗事件
    /// </summary>
    public struct CombatAction
    {
        public int ActorId;      // 攻击者ID
        public int TargetId;     // 目标ID
        public int Damage;       // 伤害值
        public bool IsSkill;     // 是否技能
    }

    /// <summary>
    /// 英雄上阵事件
    /// </summary>
    public struct HeroDeployed
    {
        public int HeroId;
        public Vector2Int Position;
        public int Level;
    }

    /// <summary>
    /// 技能释放事件
    /// </summary>
    public struct SkillUsed
    {
        public int SkillId;
        public int CasterId;
        public List<int> TargetIds;
    }

    /// <summary>
    /// 游戏状态变化事件
    /// </summary>
    public struct GameStateChanged
    {
        public Match3GameState OldState;
        public Match3GameState NewState;
    }

    /// <summary>
    /// 关卡结束事件
    /// </summary>
    public struct LevelCompleted
    {
        public bool Victory;
        public int Score;
        public int Stars;
        public List<Reward> Rewards;
    }
}

/// <summary>
/// 使用事件的系统示例
/// </summary>
public class Match3GameController : MonoBehaviour
{
    private void Start()
    {
        // 订阅事件
        EventBus.Subscribe<Match3Events.MatchFound>(OnMatchFound);
        EventBus.Subscribe<Match3Events.CombatAction>(OnCombatAction);
    }

    private void OnDestroy()
    {
        // 取消订阅
        EventBus.Unsubscribe<Match3Events.MatchFound>(OnMatchFound);
        EventBus.Unsubscribe<Match3Events.CombatAction>(OnCombatAction);
    }

    private void OnMatchFound(Match3Events.MatchFound e)
    {
        // 处理消除逻辑
        ProcessMatch(e);

        // 触发战斗
        if (IsHeroGem(e.GemId))
        {
            EventBus.Publish(new Match3Events.CombatAction
            {
                ActorId = GetHeroId(e.GemId),
                TargetId = GetCurrentEnemyId(),
                Damage = CalculateDamage(e.MatchCount),
                IsSkill = false
            });
        }
        else if (IsSkillGem(e.GemId))
        {
            EventBus.Publish(new Match3Events.SkillUsed
            {
                SkillId = GetSkillId(e.GemId),
                CasterId = GetActiveHeroId(),
                TargetIds = GetEnemyIds()
            });
        }
    }

    private void OnCombatAction(Match3Events.CombatAction e)
    {
        // 播放战斗动画
        PlayCombatAnimation(e);

        // 应用伤害
        ApplyDamage(e.TargetId, e.Damage);
    }
}

/// <summary>
/// UI系统响应事件
/// </summary>
public class Match3UIController : MonoBehaviour
{
    [SerializeField] private Text scoreText;
    [SerializeField] private GameObject victoryPanel;
    [SerializeField] private GameObject defeatPanel;

    private void OnEnable()
    {
        EventBus.Subscribe<Match3Events.GameStateChanged>(OnGameStateChanged);
        EventBus.Subscribe<Match3Events.LevelCompleted>(OnLevelCompleted);
    }

    private void OnDisable()
    {
        EventBus.Unsubscribe<Match3Events.GameStateChanged>(OnGameStateChanged);
        EventBus.Unsubscribe<Match3Events.LevelCompleted>(OnLevelCompleted);
    }

    private void OnGameStateChanged(Match3Events.GameStateChanged e)
    {
        // 更新UI状态
        switch (e.NewState)
        {
            case Match3GameState.Playing:
                // 显示游戏UI
                break;
            case Match3GameState.Paused:
                // 显示暂停菜单
                break;
            case Match3GameState.GameOver:
                // 显示结算界面
                break;
        }
    }

    private void OnLevelCompleted(Match3Events.LevelCompleted e)
    {
        if (e.Victory)
        {
            victoryPanel.SetActive(true);
            // 显示奖励等
        }
        else
        {
            defeatPanel.SetActive(true);
        }
    }
}

/// <summary>
/// 音效系统响应事件
/// </summary>
public class Match3AudioController : MonoBehaviour
{
    [SerializeField] private AudioClip matchSound;
    [SerializeField] private AudioClip skillSound;
    [SerializeField] private AudioClip victorySound;

    private AudioSource audioSource;

    private void OnEnable()
    {
        audioSource = GetComponent<AudioSource>();

        EventBus.Subscribe<Match3Events.MatchFound>(OnMatchFound);
        EventBus.Subscribe<Match3Events.SkillUsed>(OnSkillUsed);
        EventBus.Subscribe<Match3Events.LevelCompleted>(OnLevelCompleted);
    }

    private void OnDisable()
    {
        EventBus.Unsubscribe<Match3Events.MatchFound>(OnMatchFound);
        EventBus.Unsubscribe<Match3Events.SkillUsed>(OnSkillUsed);
        EventBus.Unsubscribe<Match3Events.LevelCompleted>(OnLevelCompleted);
    }

    private void OnMatchFound(Match3Events.MatchFound e)
    {
        audioSource.PlayOneShot(matchSound);
    }

    private void OnSkillUsed(Match3Events.SkillUsed e)
    {
        audioSource.PlayOneShot(skillSound);
    }

    private void OnLevelCompleted(Match3Events.LevelCompleted e)
    {
        if (e.Victory)
        {
            audioSource.PlayOneShot(victorySound);
        }
    }
}
```

## 本课小结

### 核心知识点

| 知识点 | 核心要点 |
|--------|----------|
| 观察者模式 | 一对多依赖，状态变化通知 |
| C# delegate | 类型安全的函数指针 |
| C# event | 封装delegate，限制外部访问 |
| UnityEvent | Inspector可配置，适合策划使用 |
| EventBus | 全局事件管理，解耦系统 |
| ScriptableObject事件 | 跨场景通信，资源化配置 |

### 事件系统选择指南

| 场景 | 推荐方案 |
|------|----------|
| 简单回调 | C# Action/Func |
| 需要Inspector配置 | UnityEvent |
| 跨系统通信 | EventBus |
| 跨预制体/场景 | ScriptableObject事件 |
| 高频调用 | C# event（避免UnityEvent） |

### 内存泄漏预防

```csharp
// ⚠️ 危险：忘记取消订阅
EventBus.Subscribe<SomeEvent>(OnSomeEvent);
// 如果不取消订阅，对象无法被GC回收

// ✅ 安全：使用OnDisable/OnDestroy
private void OnEnable() => EventBus.Subscribe<SomeEvent>(OnSomeEvent);
private void OnDisable() => EventBus.Unsubscribe<SomeEvent>(OnSomeEvent);
```
