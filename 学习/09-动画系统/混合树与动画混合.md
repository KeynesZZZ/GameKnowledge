# 混合树与动画混合

> 第2课 | 动画系统模块

## 1. 混合树基础

### 1.1 什么是混合树？

混合树（Blend Tree）用于根据参数值**平滑混合多个动画**。

```
速度参数: 0 ────────────────→ 1
动画:     待机 ─────混合────→ 跑步
```

### 1.2 创建混合树

```csharp
using UnityEngine;

public class BlendTreeController : MonoBehaviour
{
    private Animator animator;

    private void Start()
    {
        animator = GetComponent<Animator>();
    }

    private void Update()
    {
        // 获取移动输入
        float horizontal = Input.GetAxis("Horizontal");
        float vertical = Input.GetAxis("Vertical");

        // 设置混合参数
        animator.SetFloat("Horizontal", horizontal);
        animator.SetFloat("Vertical", vertical);

        // 计算速度
        float speed = new Vector2(horizontal, vertical).magnitude;
        animator.SetFloat("Speed", speed);
    }
}
```

## 2. 1D 混合树

### 2.1 单参数混合

```
速度: 0 → 0.5 → 1.0
动画: 待机 → 走 → 跑
```

**设置步骤**：
1. 创建 Blend Tree
2. 选择 Blend Type: 1D
3. 添加 Motion（动画片段）
4. 设置阈值

### 2.2 代码控制

```csharp
public class MovementBlend : MonoBehaviour
{
    public float walkSpeed = 2f;
    public float runSpeed = 6f;
    public float acceleration = 10f;

    private Animator animator;
    private float currentSpeed;

    private void Start()
    {
        animator = GetComponent<Animator>();
    }

    private void Update()
    {
        // 目标速度
        float targetSpeed = Input.GetKey(KeyCode.LeftShift) ? runSpeed : walkSpeed;

        // 平滑过渡
        currentSpeed = Mathf.Lerp(currentSpeed, targetSpeed, acceleration * Time.deltaTime);

        // 设置归一化速度 (0-1)
        float normalizedSpeed = Mathf.InverseLerp(0, runSpeed, currentSpeed);
        animator.SetFloat("Speed", normalizedSpeed);
    }
}
```

## 3. 2D 混合树

### 3.1 方向 + 速度混合

```
          Forward (0, 1)
              │
    Left ────┼──── Right
  (-1, 0)    │     (1, 0)
              │
          Back (0, -1)
```

### 3.2 2D Simple Directional

```csharp
public class DirectionalMovement : MonoBehaviour
{
    private Animator animator;
    public float dampTime = 0.1f;

    private void Start()
    {
        animator = GetComponent<Animator>();
    }

    private void Update()
    {
        float horizontal = Input.GetAxis("Horizontal");
        float vertical = Input.GetAxis("Vertical");

        // 使用 damp 平滑过渡
        animator.SetFloat("Horizontal", horizontal, dampTime, Time.deltaTime);
        animator.SetFloat("Vertical", vertical, dampTime, Time.deltaTime);
    }
}
```

### 3.3 2D Freeform Directional

**适用于**：任意方向移动，包括对角线

```
动作位置设置:
(-1, 0, LeftWalk)
(1, 0, RightWalk)
(0, 1, ForwardWalk)
(0, -1, BackWalk)
(-0.7, 0.7, ForwardLeftWalk)  // 对角线
(0.7, 0.7, ForwardRightWalk)
```

### 3.4 2D Freeform Cartesian

**适用于**：角色朝向摄像机时的左右前后移动

```
X轴 = 左右移动
Y轴 = 前后移动（朝向/远离摄像机）
```

## 4. 直接混合（Direct Blend）

### 4.1 手动权重控制

```csharp
public class DirectBlendController : MonoBehaviour
{
    private Animator animator;
    private int blendParamA = Animator.StringToHash("BlendA");
    private int blendParamB = Animator.StringToHash("BlendB");

    private void Start()
    {
        animator = GetComponent<Animator>();
    }

    public void SetBlendWeights(float weightA, float weightB)
    {
        animator.SetFloat(blendParamA, weightA);
        animator.SetFloat(blendParamB, weightB);
    }

    // 示例：受伤动作混合
    public void ApplyDamage(float intensity)
    {
        // 正常动画权重降低，受伤动画权重提高
        float damageWeight = Mathf.Clamp01(intensity);
        SetBlendWeights(1 - damageWeight, damageWeight);
    }
}
```

## 5. 混合树最佳实践

### 5.1 阈值设置

```
❌ 错误：不均匀阈值
0, 0.3, 1.0  → 中间过渡不平滑

✅ 正确：均匀阈值
0, 0.5, 1.0  → 平滑过渡
```

### 5.2 动画同步

```csharp
// 确保混合的动画长度相近
// 或者使用动画速度调整

public class AnimationSync : MonoBehaviour
{
    private Animator animator;

    public void SetAnimationSpeed(float targetLength)
    {
        // 获取当前动画长度
        AnimatorStateInfo state = animator.GetCurrentAnimatorStateInfo(0);
        float currentLength = state.length;

        // 计算速度倍率
        float speedMultiplier = currentLength / targetLength;
        animator.SetFloat("AnimationSpeed", speedMultiplier);
    }
}
```

### 5.3 平滑过渡

```csharp
public class SmoothBlending : MonoBehaviour
{
    private Animator animator;

    [Header("Damp Times")]
    public float speedDampTime = 0.1f;
    public float directionDampTime = 0.2f;

    private void Update()
    {
        float speed = CalculateSpeed();
        Vector2 direction = CalculateDirection();

        // 使用不同 damp 时间
        animator.SetFloat("Speed", speed, speedDampTime, Time.deltaTime);
        animator.SetFloat("DirectionX", direction.x, directionDampTime, Time.deltaTime);
        animator.SetFloat("DirectionY", direction.y, directionDampTime, Time.deltaTime);
    }

    private float CalculateSpeed()
    {
        return new Vector2(Input.GetAxis("Horizontal"), Input.GetAxis("Vertical")).magnitude;
    }

    private Vector2 CalculateDirection()
    {
        return new Vector2(Input.GetAxis("Horizontal"), Input.GetAxis("Vertical"));
    }
}
```

## 6. 高级混合技巧

### 6.1 多层混合树

```csharp
public class LayeredBlending : MonoBehaviour
{
    private Animator animator;

    // 基础移动层
    public void UpdateMovement(float speed, Vector2 direction)
    {
        animator.SetFloat("Speed", speed);
        animator.SetFloat("DirX", direction.x);
        animator.SetFloat("DirY", direction.y);
    }

    // 叠加动作层（如受伤、疲劳）
    public void ApplyFatigue(float fatigueLevel)
    {
        // 调整叠加层权重
        animator.SetLayerWeight(1, fatigueLevel);
    }
}
```

### 6.2 运行时创建混合树

```csharp
using UnityEditor.Animations;

public class RuntimeBlendTree
{
    public BlendTree CreateBlendTree(AnimationClip idle, AnimationClip walk, AnimationClip run)
    {
        BlendTree blendTree = new BlendTree();
        blendTree.blendType = BlendTreeType.Simple1D;
        blendTree.blendParameter = "Speed";

        blendTree.AddChild(idle, 0f);
        blendTree.AddChild(walk, 0.5f);
        blendTree.AddChild(run, 1f);

        return blendTree;
    }
}
```

## 本课小结

### 混合树类型对比

| 类型 | 参数 | 适用场景 |
|------|------|----------|
| 1D | 1个 | 速度变化 |
| 2D Simple Directional | 2个 | 方向移动 |
| 2D Freeform Directional | 2个 | 任意方向 |
| 2D Freeform Cartesian | 2个 | 摄像机相对 |
| Direct | N个 | 手动权重 |

### 最佳实践

1. **动画长度相近** - 避免混合不同步
2. **均匀阈值** - 确保平滑过渡
3. **使用 Damp** - 平滑参数变化
4. **合理分层** - 基础+叠加动画
